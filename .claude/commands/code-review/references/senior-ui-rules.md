# 시니어 친화 UI 상세 규칙

## 터치 타겟
- 최소 52x52px (WCAG 2.5.8 기준보다 엄격)
- 인접 타겟 간 최소 8px 간격
- 체크박스/라디오 버튼: 라벨 영역 포함 52px 이상

## 타이포그래피
- 본문: 18px 베이스 (1.6 line-height)
- 제목: 24px 이상
- caption/배지만 15px 허용
- 폰트: Pretendard Variable (next/font/local)
- 행간: 최소 1.5

## 색상/대비
- 브랜드 코랄: #FF6F61 (--color-primary)
- 텍스트/배경 대비: 4.5:1 이상 (WCAG AA)
- 링크: 밑줄 또는 명확한 색상 구분

## 버튼
- 높이: 52px(모바일) / 48px(데스크탑)
- 패딩: 최소 16px 좌우
- 글쓰기 FAB: 플로팅, 항상 보이는 위치

## 모달/시트
- 모바일: 하단 풀스크린 시트 (bottom sheet)
- 데스크탑: 중앙 팝업
- 닫기 버튼: 명확하고 크게

## 금지 용어
- "시니어", "액티브 시니어" 절대 금지
- 대체: "우리 또래", "50대 60대", "인생 2막"
