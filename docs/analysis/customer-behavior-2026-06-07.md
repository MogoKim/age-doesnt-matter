## 1. Executive Summary — 3대 발견

### 발견 ① 실고객은 63명 (6/4 46명 → 3일간 +17명), 재방문 24%
진짜 카카오 가입자 63명. 재방문(활동일 2일+) 15명(24%), 가입 후 이탈 60%, 온보딩 미완 이탈 10%.
6/4(46명, 재방문 22%) 대비 모수·재방문율 모두 소폭 상승했으나 **구조는 동일** — 4명 중 3명이 가입하고 안 돌아온다.

### 발견 ② 유입 채널 효율이 30배 차이 — Kakao가 압도적, 직접입력·YouTube는 깔때기 누수
| 채널 | 세션 | 가입전환 | 재방문율 |
|------|------|---------|---------|
| **Kakao** | 43 | **30%** | **26%** |
| Google | 77 | 12% | 3% |
| TWA 앱 | 564 | 4% | 10% |
| Naver | 200 | 2% | 3% |
| 직접입력 | 1,871 | 1% | 1% |
| YouTube | 343 | 1% | 4% |

→ 트래픽 대부분(직접입력 1,871·YouTube 343)이 **전환·재방문 최저 채널**. Kakao는 적게 오지만 가입·정착 모두 최고.

### 발견 ③ "가입 전 정독 → 재방문", "첫 활동 → 재방문" 인과가 더 강해졌다
- 가입 전 콘텐츠 정독(📖): 8명 → 재방문 **63%** / 정독 안 함 55명 → 18%
- 글 쓴 유저 재방문 33%, 댓글 쓴 유저 31% vs **무활동 4%** (8배)
- Aha Moment: 재방문 유저는 온보딩 94%·글작성 31%·댓글 25%, 1회성은 온보딩 44%·글 4%·댓글 4%

→ **콘텐츠로 마음이 동해 가입하고, 첫 활동을 한 사람이 남는다.** 6/4 결론과 일관 + 표본 증가로 신뢰도 강화.

### 한 줄 처방
> **(획득) 직접입력·YouTube 트래픽을 늘리기보다 Kakao·Google 같은 고전환 채널에 집중하고,**
> **(정착) 가입 전 좋은 글 노출 → 가입 직후 첫 활동(댓글) 유도 동선을 강화한다.**

## 2. 4대 초점별 결론

### 2-1. 재방문 / 리텐션
- **회원 코호트**: 활동일 2일+ 재방문 15/63명(24%). 가입 후 이탈 60%.
- **세션 코호트**(비회원 포함, sessionId 기준 — 신뢰): 전체 3,731세션 중 **96%가 1일만 방문 후 이탈**, 재방문 4%. 첫방문주 D7 재방문 0~1%.
- **login 기반**(5/15+): login 34명 중 11명(32%)이 2일+ 로그인 → 가입까지 간 사람의 1/3은 다시 온다.
- 해석: 깔때기 맨 위(첫 방문)에서 96% 증발. 가입 단계를 넘으면 리텐션이 급등(4%→32%) → **가입 전환이 정착의 병목**.

### 2-2. 가입 전환 퍼널 (최근 30일, 세션 기준 — 신뢰)
```
방문 3,130
  → 카카오 버튼 클릭 107 (3%)
  → 로그인 완료 67 (2%)
  → 가입절차(signup_step) 51
  → 글 작성 4 (0.1%)
```
- **방문 → 카카오 버튼 3%**가 최대 누수. 97%가 로그인 시도조차 안 함.
- 온보딩 단계: step1 33 → step2 29 → step3 18세션 (step1→3에서 약 45% 이탈).
- `signup_abandoned` 5건 전원 step1 중단(소요 0.5초~162초) — step1 허들 존재.
- 이탈 직전 페이지: 홈(`/`) 49%, 매거진 15%.

### 2-3. 콘텐츠 참여
- USER 작성 글: STORY 18 · JOB 5 · HUMOR 4 · MAGAZINE 4 · WEEKLY 2 (총 33개).
- **읽기 깊이**(PostView 186건): 0~24%에서 80% 이탈, 완독 1%. → 글 도입부에서 대부분 떠남.
- USER 댓글은 거의 전부 **BOT 글**(STORY/BOT 등)에 달림. USER 글에 달린 USER 댓글은 13개뿐 → 실고객 간 상호작용 미미, 봇 콘텐츠가 댓글 유발의 주력.

### 2-4. 유입 채널 효율
- 발견 ②표 참조. **Kakao(30%/26%) ≫ Google(12%) ≫ TWA앱(4%/10%) ≫ Naver(2%) ≫ 직접입력·YouTube(1%)**.
- TWA 앱(564세션)은 전환 낮으나 재방문 10%로 2위 — 이미 설치한 사람은 다시 옴.
