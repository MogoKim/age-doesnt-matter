✅ 창업자 결정 반영
  • 이미 가입인사 남긴 사람 → B 위젯 안 뜸 (가입인사 작성 여부로 판별)

  • 비회원 → B 위젯 안 뜸 (회원 전용 — 비회원은 신입환영 섹션만)
  • 신입환영 섹션은 계속 떠야 맞습니다 — 매일 "오늘 온 신입"이 바뀌는 콘텐츠라 거슬리지 않고, 끄면 실유저 환대가 안 됩니다.
🔔 글 쓰면 "푸시 알림 받기" 요청 — 이미 작동 (추가 개발 0) 회원이 가입인사를 쓰면 "답글 오면 바로 알려드릴까요?" 푸시 동의 요청이 자동으로 뜹니다(PostWriteForm:332 이미 있음). → 환대 댓글이 오면 푸시 알림 → 다시 들어옴(D1↑). 가입인사에도 그대로 적용됩니다.
🛡️ 비회원 — 언제·어디서 비회원은 "가입 직후"라는 시점이 없으니 B 위젯이 없습니다. 입구는 홈 신입환영 섹션의 "나도 인사하기"(상시) + 사는이야기 가입인사 탭. → 가입 안 해도 언제든 첫 인사(Turnstile).
별도 비회원 전용 배너는 안 띄웁니다 — 이미 가입 유도가 3겹(정체성배너·인라인·띠배너)이라 충돌·피로를 줍니다.
