6

디멘드젠(여성 45~70 웹) 처방

  • 🔴 그냥 웹 홈/사연/매거진으로 랜딩시키면 D1 1~2% — 광고비 대부분 손실. 현재 Google 유입(68세션)도 D1 2.9%에 그침.
  • ✅ 랜딩을 "베스트"로 — 웹에서 유일하게 D1 12% 콘텐츠. 인기글 모음으로 첫인상 = 둘러볼 거리.
  • ✅ "3화면 이상 둘러보게" — 웹 매직넘버는 3화면. 관련글·다음글·인기글 동선을 베스트/사연 하단에 강하게.
  • 🚫 매거진·일자리 랜딩 금지 — D1 0~2.7%, 막다른 길.
  • 💡 중장기: 웹 유입을 앱 설치로 — 진짜 리텐션(앱 12%)은 앱에서 나온다. 웹에서 잡은 뒤 앱으로 넘기는 동선(현재 설치유도 OFF)을 켜는 것도 핵심.
요약 처방: 디멘드젠 → 베스트 랜딩 → 3화면+ 둘러보기 → (여유되면) 앱 설치 유도. 이 3단계가 웹 유입을 잔존시키는 데이터 기반 경로.
