## 7. 내부링크 구조안

```
[갱년기 허브]  ──▶ 갱년기 매거진 15 (증상별)
     │                    │
     │                    ▼
     │            매거진 하단 "실제 경험담" ──▶ 커뮤니티 경험글(증상 질문/관리 성공담)
     │                                              │
     ▼                                              ▼
홈/매거진 목록에서 허브 노출          커뮤니티 글 하단 "관련 정보" ──▶ 관련 매거진
```

- **허브 → 매거진**: 갱년기 허브에서 15개 매거진을 증상 카테고리(열감/불면/질건조증/체중/관계)로 그룹 링크
- **매거진 → 커뮤니티 경험글**: 각 매거진 하단에 같은 증상 커뮤니티 경험담 2~3개(예: 질건조증 매거진 → "갱년기 질건조증 참지마세요" 경험글)
- **커뮤니티 경험글 → 관련 매거진**: 증상 질문글 하단에 해당 매거진 1개 추천(기존 관련글 추천 v2 로직 활용 가능)
- **홈/매거진 목록에서 밀 것**: "50대 여성 갱년기 증상" 허브 + 클릭 실적 있는 "퇴직 후 외로움" 매거진

## 8. 리스크 / 주의사항

1. **YMYL(건강) 영역**: 갱년기·폐경·질건조증·호르몬치료는 의학 정보. **의학적 단정 금지, "증상 시 산부인과 상담" 문구 필수.** (P0 종합감기약 seoDesc처럼 "참고용·상담 권유" 톤)
2. **성적 표현 (AdSense 정책)**: 질건조증·부부관계·속궁합·성욕 등은 광고 정책상 민감. 매거진은 의료 톤 유지 시 OK, 커뮤니티 자극 제목은 허브에서 링크 제외.
3. **www / non-www 중복**: GSC에 `www.age-doesnt-matter.com/magazine/...` 색인 혼입 확인(퇴직후외로움·갱년기몸신호). canonical/리다이렉트 정규화 필요 — 별도 기술 과제.
4. **?from=trending 파라미터 중복 색인**: 다수 커뮤니티 URL이 trending 쿼리판으로 색인(P0 글6과 동일). canonical 무시되는 중복.
5. **제목 카니벌라이제이션**: 갱년기 "몸이 보내는 신호/증상별 대처법" 유사 매거진 5~6개가 서로 순위 경쟁.
6. **"시니어" 금지어**(CLAUDE.md): 허브 카피는 "50·60대", "우리 또래", "인생 2막"으로.

## 정정 이력 (2026-07-06)

> **정정**: 초판에서 "부부 친밀감 회복 매거진 = 미색인(unknown)"으로 기재했으나 **오판이었다. 실제로는 색인됨(Submitted and indexed)**.

- **오판 원인**: URL Inspection 조회 시 slug를 `...관계-변화-대처법-5`로 **절삭해서** 넣어(정확 slug는 `...관계-변화-대처법-5가지`) 존재하지 않는 URL을 조회 → "unknown" 반환. 정확한 full slug로 재조회하니 정상 색인 확인.
- **부부 친밀감 회복 정정 후 상태**:
  - 정확 URL: `/magazine/50대-여성-부부-친밀감-회복-갱년기-이후-관계-변화-대처법-5가지` (postId `cmr5s7obq0002f02y64ou2yq8`)
  - GSC: **Submitted and indexed**, 2026-07-04 크롤, pageFetch SUCCESSFUL, google=user canonical 일치
  - robots `index, follow` / sitemap 포함 / HTTP 200 (www→308 정규화) / seoTitle·seoDescription 존재
  - → 색인 요청 **불필요**. 순위 부양(내부링크)만 필요.
- **실제 미색인은 빈둥지 증후군 1개뿐**:
  - URL: `/magazine/자녀-독립-후-허전함-빈-둥지-증후군-극복법-5가지` (postId `cmr8n2j4d0002rr2yfv2e5m0t`)
  - 사유: 2026-07-06 발행(신규)이라 아직 미크롤(lastCrawl null). 기술 상태는 완벽(200·canonical self·robots index·sitemap 포함·seo 메타 완비).
  - → 색인 요청 유효·안전. 단 **non-www canonical URL로 요청**([magazine-generator.ts:873](../../agents/cafe/magazine-generator.ts#L873) fallback이 www이므로 주의).
- **교훈**: URL Inspection·GSC 조회는 slug를 절삭·추정하지 말고 DB의 정확한 full slug로 조회할 것.
