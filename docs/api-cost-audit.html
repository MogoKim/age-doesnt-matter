6. 비용 절감 추가 포인트

즉시 가능 — ~$1.5/월 절감

DALL-E 카드뉴스 비활성화

agents/cmo/card-news/image-gen.ts에서 DALL-E 호출이 남아있음. Cafe 이미지 생성기처럼 Gemini Playwright 이관 또는 Unsplash 폴백으로 전환하면 OpenAI 비용 거의 0으로 감소. 현재 월 ~$1.5 소비 중.

즉시 가능 — 충전 빈도 감소

Anthropic 자동충전 금액 $15 → $50 상향

현재 잔액 $5 이하 → $15 충전 (5~6일마다 충전, 월 ~5회). $5 이하 → $50 충전으로 변경 시 한 번 충전으로 약 25일 사용 가능. 충전 빈도 감소로 관리 부담 줄어듦. 비용 자체는 동일. 설정: console.anthropic.com → Billing → Auto-recharge.

중기 검토

Gemini 구독 vs 실제 TTS 사용량 비교

₩11,000/월(~$8) 구독 중. 주 사용: 영상 나레이션 TTS (수동 실행). Gemini TTS API 단가는 $0.10/1M chars — 60초 나레이션 약 300자 = $0.00003/편. 월 100편도 $0.003. 구독 필요성 재검토 권장. (단, Gemini Imagen 스크래핑을 Google One 구독으로 사용 중이면 유지 필요)

중기 검토 — ~$15/월 절감 가능

단순 에이전트 Sonnet → Haiku 다운그레이드

단순 판단/집계 작업 에이전트 일부를 Haiku로 전환. Haiku는 Sonnet 대비 약 10분의 1 비용. BotLog 분석으로 Haiku 전환 가능 태스크 식별 후 순차 적용.

7. API 장애 시 영향도

ℹ️
각 API가 완전히 중단되면 어떤 일이 벌어지는지 정리. 우선순위 높은 항목부터 확인.
API즉각 영향 (0~1h)누적 영향 (1일+)사용자 체감자동 복구
Supabase DB 모든 페이지 500. 로그인 불가. 에이전트 전부 중단 서비스 완전 마비 사이트 자체가 열리지 않음 ❌
Anthropic Claude 에이전트 전체 콘텐츠 생성 중단. Slack #긴급 알림 매거진 공백, 시드봇 조용, SNS 포스팅 중단 "새 글이 안 올라온다" ❌ 충전 필요
Vercel (배포) 신규 배포 불가. 기존 서비스는 CDN 캐시로 유지 코드 배포 중단 현행 서비스는 정상. 신기능 불가 ✅ CDN 캐시
Slack 알림 전송 실패. 에이전트는 계속 실행됨 오류 발생해도 모름 — 모니터링 블라인드 내부적으로만. 사용자는 정상처럼 느낌 ❌
Cloudflare R2 이미지/영상 업로드 실패. 기존 이미지는 정상 신규 매거진 이미지 없음, 프로필 업로드 불가 매거진 이미지 깨짐 (신규 글만) ❌
Instagram/Threads SNS 포스팅 실패. BotLog FAILED 소셜 채널 침묵 SNS에 글이 안 올라옴 ❌ 재시도 없음
X (Twitter) 429 에러 시 즉시 실패 X 채널 침묵 X에 글이 안 올라옴 ❌ 재시도 없음
Google Indexing 색인 요청 실패 검색 노출 지연 (며칠~1주) 신규 글이 검색에 잘 안 잡힘 ❌
Upstash Redis Rate limit 체크 실패 → 에이전트 실행 가능성 API 중복 호출 위험 이상 없으나 내부 안전장치 해제 ⚠️ 부분
OpenAI DALL-E 카드뉴스 이미지 생성 실패 카드뉴스 이미지 없음 SNS 카드뉴스 품질 하락 수동 대응
Kakao OAuth 신규 로그인 불가. 기존 세션은 유지 로그아웃 후 재로그인 불가 "카카오 로그인이 안 됩니다" ❌ 카카오 측 복구 필요
Gemini TTS 영상 나레이션 생성 실패 (수동이라 즉각 영향 없음) 무음 영상 영상 품질 하락 수동 프로세스
Facebook / Band 현재 비활성화 상태 — 영향 없음 (Facebook 코드 차단, Band API 심사중)
