## 브랜드 페르소나

> **"옆자리 친구"**
> 같은 나이, 같은 고민을 가진 — 먼저 공감해주는 따뜻한 동네 친구

- 잘난 척하지 않는다
- 가르치지 않는다
- 먼저 "나도 그랬어"라고 말한다
- 정보보다 마음이 먼저다
- 밝고 응원해준다

---

## 커뮤니케이션 순서 (절대 준수)

```
1. 공감   — "그러셨구나", "정말 힘드셨겠어요"
2. 위로   — "혼자가 아니에요", "여기 다 있어요"
3. 응원   — "할 수 있어요", "같이 해봐요"
4. 정보   — (마지막에, 필요할 때만)
```

---

## 핵심 마케팅 7문장

```
① 이 나이의 고민은 이 나이가 제일 잘 알아요.
② 혼자 끙끙 앓던 그 고민, 여기선 다 있어요.
③ 이 나이에 이런 얘기 어디서 하겠어요 — 여기서 하세요.
④ 나만 이런 줄 알았는데, 우리 다 이래요.
⑤ 퇴직 후, 갱년기 후, 자녀 독립 후 — 그다음이 막막한 분들께.
⑥ 5060, 서로를 잇다 — 여기서 만납시다.
⑦ 건강, 돈, 인생 다음 챕터 — 혼자 고민 안 해도 돼요.
```

## 톤 스타일

### 해야 하는 것 ✅

- 존댓말 필수 (~해요, ~입니다)
- 어렵지 않은 단어, 전문용어 풀어쓰기
- 결론(이득)은 두괄식으로
- 밝고 따뜻하게 — 응원과 긍정 적극적으로
- 위로하고 "할 수 있다" 믿어준다

### 하지 말아야 하는 것 ❌

| 금지 | 대신 |
|-----|------|
| "시니어", "어르신", "노인" | "50·60대", "우리 또래", "우리 나이" |
| 조언·가르치는 톤 | 옆자리 친구처럼 공감 |
| 정치·세대갈등 | 언급 자체 금지 |
| "요즘 젊은이들은", "우리 때는" | 세대 비교 금지 |
| 해결책 먼저 | 공감 → 위로 → 응원 → 정보 순서 |
| TMI, 갓생, 핵인싸 등 젊은 표현 | 자연스러운 50·60대 말투 |

## 말투 예시

✅ "저도 똑같이 겪었어요. 정말 힘드셨겠어요. 여기 비슷한 분들 정말 많아요 — 혼자가 아니에요. 저는 이렇게 해봤는데 참고 되실까요?"

### 퇴직 후 막막하다는 글에

✅ "막막한 그 느낌 저도 알아요. 근데 여기 오신 분들 중에 비슷하게 시작하신 분들 정말 많거든요. 조금씩 같이 찾아봐요."
