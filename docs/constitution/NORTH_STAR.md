# 우나어 북극성 문서 (NORTH STAR)

## 우리 나이가 어때서: 일과 삶을 잇다

> **이 문서는 모든 의사결정의 기준입니다.**
> 기능이 바뀌고, 디자인이 바뀌고, 팀이 바뀌어도 이 문서는 바뀌지 않습니다.
> AI 에이전트도, 개발자도, 창업자도 이 기준을 먼저 읽습니다.

**문서 버전**: v3.1
**최종 수정일**: 2026-03-16
**작성자**: Product Owner

> 작은 소일거리부터 하루의 이야기까지,
> 당신의 새로운 '내 일'과 든든한 '내 편'을 잇습니다.

---

## 2. 본질 (Essence)

> **제2의 인생을 위해, 50·60대의 '내 일(Job)'과 '내 편(Community)'을 잇는 커뮤니티**

| 키워드 | 의미 |
|---|---|
| **Job** | 단순한 채용이 아니라 나의 역할·소득·활동 |
| **Community** | 정보만 주는 곳이 아니라 정서적으로 기댈 수 있는 편 |
| **잇다** | 일로 들어와서, 사람으로 남는 구조 |

---

## 3. MISSION

> **50·60대가 다시 사회와 연결되고, 자신만의 속도로 새로운 삶을 시작할 수 있도록 돕는다.**

- "성공시키겠다"가 아니라 "시작할 수 있게 돕는다"
- 속도를 강요하지 않는다 — 이것이 우나어의 성격을 정의한다

---

## 4. VISION

> **나이가 가능성의 한계가 되지 않는 세상, 누구나 설레는 마음으로 두 번째 전성기를 맞이하는 라이프 플랫폼**

---

## 5. CORE VALUES

| 가치 | 정의 | 판단 기준 |
|---|---|---|
| **실용 (Utility)** | 당장 삶에 써먹을 수 있는 정보와 선택지를 제공한다 | "이 기능/콘텐츠가 당장 도움이 되는가?" |
| **따뜻함 (Warmth)** | 나이를 이유로 소외되지 않도록 사람을 먼저 생각한다 | "이 말투/UI가 사용자를 존중하고 있는가?" |
| **연결 (Connection)** | 일·일상·사람·지역을 잇는 연결을 만든다 | "사용자를 고립시키지 않는가?" |
| **용기 (Courage)** | 늦어도 괜찮다는 믿음으로 다시 시작할 힘을 준다 | "시작하기 두려운 사람에게 용기를 주는가?" |
| **존중 (Respect)** | 삶의 속도와 선택을 비교하지 않는다 | "비교·경쟁을 유발하지 않는가?" |
| **현실성 (Reality)** | 이상보다 지금의 삶에 도움이 되는 선택을 한다 | "현실과 동떨어진 조언은 아닌가?" |

### 내부 한 줄 요약

> **우나어는 '다시 시작하고 싶은 50·60대가 일과 일상을 매개로 사람과 연결되는 제2의 인생 커뮤니티'다.**

## 6. 타겟 정의

### 비율

| 구분 | 비율 | 설명 |
|---|---|---|
| **40대 후반~60대 여성** | 100% | 핵심 타겟 |

### 4가지 페르소나 (우선순위 순)

#### 1순위 — B타입: 외로움 해소형 (핵심 메인)

> "말동무가 필요해. 내 얘기 좀 들어줘."

- **주요 연령/성별**: 50대 여성
- **핵심 결핍**: 자녀 독립 후 빈 둥지 증후군, 대화 상대 부재
- **욕구**: 평가 없이 들어주고 맞장구쳐 주는 안전한 공간
- **주 이용 메뉴**: 사는 이야기 (/community/stories), 활력 충전소 (/community/humor)
- **운영 원칙**: **"안전하고 따뜻한 수다방을 사수하라."** — 정치/혐오 글 차단이 이 타겟의 생명줄
- **성공 지표**: 글·댓글 작성 빈도, 재방문율

#### 2순위 — D타입: 실리 추구형

> "정부 지원금이나 쏠쏠한 정보라도 놓치면 억울하지."

- **주요 연령/성별**: 60대 여성
- **핵심 결핍**: 정보의 소외, 소득 창출 욕구, 신체적 노화
- **욕구**: 돈 되는 실용 정보, 놓치면 손해 보는 것들
- **주 이용 메뉴**: 매거진 (/magazine), 내 일 찾기 (/jobs)
- **운영 원칙**: **"복잡한 말 치우고, 확실하게 이득이 되는 알짜 정보를 직관적으로."**
- **성공 지표**: 매거진 조회수, 스크랩수, 일자리 클릭률

#### 3순위 — C타입: 정보 탐색형

> "좋은 정보 놓치면 억울해. 내가 직접 찾아봐야지."

- **주요 연령/성별**: 50대 후반~60대 여성
- **핵심 결핍**: 사회적 변화에 뒤처지는 불안, 재취업·자격증 정보 부족
- **욕구**: 믿을 수 있는 실용 정보, 또래 성공 사례
- **주 이용 메뉴**: 내 일 찾기 (/jobs), 매거진 (/magazine)
- **운영 원칙**: **"복잡한 말 없이, 바로 써먹을 수 있는 알짜 정보를 또래 시선으로."**
- **성공 지표**: 매거진 조회·스크랩수, 일자리 클릭률

#### 4순위 — A타입: 생계 절실형

> "체력이 허락하는 한 무슨 일이든 할 수 있는데, 나이 60 넘었다고 안 써주네."

- **주요 연령/성별**: 60대 이상 남/여
- **핵심 결핍**: 경제적 빈곤, 나이 제한 장벽, 디지털 접근성
- **욕구**: 지원하면 바로 일할 수 있는 현실적 공고
- **주 이용 메뉴**: 내 일 찾기 (/jobs) — 경비·물류·미화 등
- **운영 원칙**: **"감성적 멘트보다, 당장 일할 수 있는 일자리를 가장 큰 글씨와 직관적인 기능으로."**
- **성공 지표**: 일자리 조회 → 지원(전화걸기) 전환율

## 8. 콘텐츠 정책 (절대 원칙)

### 허용

- 일상생활 (요리, 손주, 여행, 건강 경험담)
- 취업·일자리 정보 및 경험
- 건강 정보 (사실 기반, 경험담)
- 재테크·연금·보험 (사실 기반)
- 유머·웃음 (긍정적, 무해한)
- 공감, 위로, 응원

### 절대 금지 (Zero-Tolerance)

| 항목 | 왜 금지인가 |
|---|---|
| **정치적 발언 및 선동** | 5060 커뮤니티가 망하는 지름길. 무조건 차단 |
| **종교 갈등 유발** | 커뮤니티 분열 원인 |
| **혐오 표현** (세대·성별·지역·인종) | 안전한 공간이라는 신뢰 파괴 |
| 성인 콘텐츠 | 서비스 정체성과 무관 |
| 도박·다단계·불법 광고 | 타겟이 피해 취약계층 |
| 타인 비방·명예훼손 | 법적 리스크 + 따뜻함 파괴 |
| 의학적 확정 진단 | 위험. 상담 권유는 가능 |

### 톤앤매너

| 속성 | 기준 |
|---|---|
| **말투** | 친절한 매니저 — 관리자가 아닌, 살뜰히 챙겨주는 조카 같은 말투 (~해요, ~입니다) |
| **가독성** | 폰트 크고 시원하게, 결론(이득)은 무조건 두괄식 |
| **금지 톤** | 반말, 권위적·가르치려는 말투, 젊은 세대 중심 표현 |

## 9. 성장 전략 (Growth)

### 락인 순환 구조

```
[내 일 찾기] → 일자리 보러 왔다가
      ↓
[활력 충전소] → 유머 보고 웃고
      ↓
[사는 이야기] → 내 고민 털어놓고
      ↓
[매거진] → 운영자 진심 어린 글에 감동 → 즐겨찾기
      ↓
[베스트] → 인기글 보며 "여기 사람 많네" → 참여 유도
```

> **"미끼(Job)를 물고 들어와, 재미(Fun)에 머물고, 사람(Life)에게 정착한다."**

### 유입 채널 전략

| 채널 | 방법 | 우선순위 |
|---|---|---|
| **카카오톡** | 밴드·카톡방 바이럴, 매거진 링크 공유 | 1순위 (타겟 주 이용 채널) |
| **네이버 밴드** | 시니어 커뮤니티 크로스 포스팅 | 2순위 |
| **구글 SEO** | 매거진 검색 유입 (기초연금, 건강 등) | 3순위 (장기 자산) |
| **유튜브** | 매거진 콘텐츠 숏폼 → 사이트 유도 | Phase 2 |

### UGC 참여 유도 전략

| 단계 | 방법 |
|---|---|
| **발견** | 홈 "뜨는 이야기", 베스트 → "여기 사람들이 활발하네" |
| **관찰** | 글·댓글 읽기 (비로그인도 가능) → "나도 저런 이야기 있는데" |
| **첫 참여** | 공감 ❤️ 누르기 (가장 낮은 허들, 로그인만 필요) |
| **댓글** | "저도요~" 한 줄 댓글 → 반응 받기 → 성취감 |
| **글쓰기** | 플로팅 FAB으로 자연스럽게 유도 → 첫 글 축하 알림 |
| **정착** | 등급 승격 (🌱→🌿) → 이미지 첨부 등 기능 해금 → 충성 사용자 |

## 10. 비즈니스 모델 (단계별)

| Phase | 목표 | 수익 모델 |
|---|---|---|
| **Phase 1 (현재)** | MAU + 재방문율 집중 | 구글 애드센스 + 쿠팡 CPS (소규모, 사용자 경험 우선) |
| **Phase 2** | 트래픽 확보 + 충성 사용자 | 외부 직접 광고 확대, 네이버 밴드 연동 |
| **Phase 3** | 프리미엄·커머스 | 프리미엄 채용 상단 노출, 공동구매 (관절/건강식품 등) |

> **"신뢰가 쌓이기 전에는 지갑을 열게 하지 않는다."**
> Phase 1에서는 광고가 있되, 사용자 경험을 해치지 않는 선에서 최소화.
> 광고 정책 상세 → PRD_Final_A 참조.

## 11. 현재 KPI (Phase 1)

### 1차 KPI: 사이트 완성

- **목표**: PRD_Final_A·B·C 모든 기능 구현 + QA 통과
- **기준**: 모든 페이지 정상 동작 + 모바일 반응형 확인 + 시니어 친화 UI 검증

| 지표 | 목표 | 측정 방법 |

| D7 리텐션 | > 20% | 가입 후 7일 내 재방문 |

| 일자리 클릭 | > 50/일 | 전화걸기/지원 CTA 클릭 |

> 자동화(봇 운영)는 1차 KPI 완료 후 별도 진행. 현재 개발 범위 외.

## 13. 절대 변경 불가 (Immutable)

아래 항목은 서비스가 존재하는 한 변경하지 않는다:

- **Mission**: "50·60대가 다시 사회와 연결되고, 자신만의 속도로 새로운 삶을 시작할 수 있도록 돕는다"
- **Vision**: "나이가 가능성의 한계가 되지 않는 세상"
- **Core Values**: 실용, 따뜻함, 연결, 용기, 존중, 현실성
- **Content Policy (ABSOLUTE_ZERO)**: 정치·종교·혐오·성인·도박·비방·진단
- **Senior-First UX**: 52px 터치 타겟, 17px 최소 폰트, 존댓말

### 변경 가능한 것

- KPI 수치 (달성 후 상향)
- Phase 상태 (1 → 2 → 3)
- 에이전트 권한 범위 (automation_status)
- 메뉴 구성·순서 (사업 전략에 따라)
- 광고 정책 세부 수치 (빈도, 슬롯 추가)
