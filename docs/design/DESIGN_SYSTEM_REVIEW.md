## 2. 타이포그래피

**폰트**: Pretendard Variable (한글 최적화, 가변 폰트)

### 크기 스케일

| 이름 | 기본 크기 | 용도 |
|------|----------|------|
| caption | **15px** | 캡션, 배지, 아이콘 레이블 (최소 허용) |
| body | **18px** | 본문 텍스트 기본 |
| title | **20px** | 섹션 제목 |
| heading | **24px** | 페이지 제목 |

> 기본 행간: **1.75** (가독성 확보)
> 한글 줄바꿈: `word-break: keep-all` 적용

### 글씨 크기 3단계 조절 시스템

사용자가 직접 글씨 크기를 선택할 수 있습니다 (헤더·푸터에 토글 배치).

| 단계 | 본문 | 캡션 | 섹션 제목 | 페이지 제목 |
|------|------|------|----------|-----------|
| **기본** | 18px | 15px | 20px | 24px |
| **크게** | 20px | 17px | 24px | 28px |
| **매우 크게** | 24px | 20px | 28px | 32px |

- 레이아웃(여백, 높이)은 변하지 않음 — 텍스트·아이콘 크기만 확대
- 쿠키 저장으로 재방문 시 설정 유지

> **피드백 요청**: 3단계 조절이 충분한지, 또는 슬라이더 방식이 더 직관적인지 의견 주세요.

## 6. 아이콘 시스템

**디자인 원칙**:
- Stroke 1.75px, round cap/join
- 24×24 viewBox
- `currentColor` 색상 상속
- `filled` prop으로 outline ↔ filled 전환

**네비게이션 아이콘 6종**:

| 아이콘 | 설명 | 연결 카테고리 |
|--------|------|------------|
| IconBest | 별 모양 | 베스트 |
| IconStories | 말풍선 | 사는이야기 |
| IconLife2 | 새싹 | 2막준비 |
| IconEnergy | 번개 | 웃음방 |
| IconMagazine | 펼친 책 | 매거진 |
| IconJobs | 서류가방 | 내일찾기 |

**유틸리티 아이콘**: Search, User, Bell, Heart, Bookmark, Share, Flag, Eye, Comment, Write, Kakao 등

> **피드백 요청**: 아이콘 스타일(stroke 기반)이 타겟 연령층에 적합한지, filled 스타일이 더 인식하기 쉬울지 의견 주세요.
