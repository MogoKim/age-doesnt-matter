# Google Ads 성과 기반 소재 개선 루프

---

## 성과 판단 기준 (임계값)

| 지표 | 목표값 | 위험 기준 | 조치 |
|------|--------|----------|------|
| CTR (클릭률) | 1.5% 이상 | 1% 미만 | 소재 교체 |
| CPC (클릭당 비용) | 300원 이하 | 500원 초과 | 타겟/입찰 조정 |
| 전환율 | 3% 이상 | 1% 미만 | 랜딩페이지 점검 |
| 노출 빈도 | 5~7회 | 10회 초과 | 소재 교체 (피로도) |
| Quality Score | 7점 이상 | 5점 미만 | 광고문구 + 랜딩 개선 |

---

## 루프 실행 흐름

```
[매일 09:00 KST] ads-data-agent 실행
    ↓
전날 성과 데이터 수집 (Playwright)
    ↓
임계값과 비교
    ↓
기준 미달 캠페인 있음? ──YES──→ Slack #에이전트에 알림
    ↓ NO                          + 교체 소재 자동 생성 시작
완료 보고만                              ↓
                              창업자 승인 (Slack 버튼)
                                    ↓
                         assets/approved/ads/ 에 저장
                                    ↓
                            수동: Google Ads UI 업로드
```

## Slack 알림 포맷

```

❌ CPC: 620원 (목표 300원 초과)

원인 분석:
- 이미지 노출 빈도 12회 → 피로도 높음
- "50대 커뮤니티" 메시지 반응 낮음

제안 소재: 감성 메시지로 교체
[소재 생성 승인] [나중에]
```

---

## 소재 교체 의사결정 트리

```
CTR < 1% AND 노출 > 5만?
    → 메시지 문제 → 메시지 베리에이션 3가지 생성

CTR 정상 + CPC 높음?
    → 타겟 문제 → 타겟 세그먼트 조정 (에이전트 제안)

CTR 낮음 + 노출 빈도 높음?
    → 피로도 문제 → 완전 새 이미지 생성

Quality Score 낮음?
    → 관련성 문제 → 광고문구 + 랜딩페이지 일치도 점검
```
