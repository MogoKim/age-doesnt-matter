# 디자인 요청 템플릿 모음

> 창업자 및 C-Level 에이전트가 디자인 작업 요청 시 사용하는 표준 템플릿.
> 이 파일을 복사해서 요청하면 에이전트가 정확하게 처리한다.

---

## 1. Product Designer — 신규 기능/페이지 설계

```
"[기능명] Figma에 설계해줘.

목적: [이 기능이 왜 필요한가]
사용자 상황: [어떤 상황에서 쓰나]
필요한 화면 목록:
  1. [화면 이름] — [역할]
  2. [화면 이름] — [역할]
특이 사항: [예외 케이스, 권한 분기 등]
참고 화면: [있다면 기존 Figma 링크]

조건:
- 데스크탑(1440px) + 모바일(390px) 둘 다
- 유저 플로우 화살표 포함
- 완료 후 Figma 노드 ID 알려줘"
```

**승인 후 코딩 요청:**
```
"Figma 확인했어, 승인할게.
이 디자인 기반으로 코딩해줘: [Figma 링크]"
```

---

## 2. Product Designer — 기존 화면 수정/개선

```
"[페이지명] Figma에서 개선해줘.
현재 Figma: [링크]

변경 내용:
- [변경1]
- [변경2]

수정 후 내가 승인하면 코딩 시작해."
```

---

## 3. Product Designer — 전체 역공학 (초기화)

```
"Product Designer야, 전체 Figma 초기화해줘.
기존 코드 전부 역공학해서 Figma에 그려줘.

순서:
1. Design System (DESIGN.md → Figma Variables)
2. 공통 컴포넌트 (src/components/ui/)
3. 페이지별 화면 (홈, 게시판, 상세, 마이, 온보딩)
4. User Flow 화살표

완료 후 Figma 파일 링크 알려줘."
```

## 4. Graphic Designer — 구글 애즈 이미지 소재

```
"구글 애즈 이미지 소재 만들어줘.

캠페인 목적: [인지도 / 전환 / 리타겟]
타겟: [50대 여성 / 은퇴 준비층 / 60대 남성]
메인 메시지 3가지:
  1. [이성적 메시지] — 예: "50대가 선택한 커뮤니티"
  2. [감성적 메시지] — 예: "우리, 서로가 있잖아"
  3. [행동 유도] — 예: "지금 가입하고 우또래 만나기"
이미지 스타일: [실사 / 일러스트 / 텍스트 위주]
AI 티 안 나게: photorealistic, candid, imperfect 적용해줘

출력 사이즈 (각 메시지 × 3사이즈 = 총 9개):
  - 1200×628 (구글 배너)
  - 300×250 (직사각)
  - 160×600 (세로 배너)

저장: assets/generated/ads/[오늘날짜]_[캠페인명]/"
```

## 5. Graphic Designer — SNS / 매거진 이미지

```
"[인스타그램 / 페이스북 / 매거진] 이미지 만들어줘.

주제: [게시글 제목 또는 주제]
브랜드 톤: 따뜻하고 친근함, 코랄(#FF6F61) 포인트
이미지 스타일: [실사 / 일러스트]
  - 실사 시: Korean woman/man 50s-60s, natural, candid shot
  - 일러스트 시: warm illustration, soft colors, inclusive

사이즈:
  - 인스타그램 피드: 1080×1080
  - 인스타그램 스토리/릴스: 1080×1920
  - 페이스북 커버: 1200×628
  - 매거진 썸네일: 800×450

저장: assets/generated/sns/[오늘날짜]_[주제]/"
```

## 6. Video Director — 광고/콘텐츠 영상

```
"[초수]초 영상 만들어줘.

주제/메시지: [핵심 내용]
타겟: [50대 여성 등]
스타일: [오디오북형 / 정보형 / 감성형]
나레이션: 한국어 여성, 따뜻하고 차분한 목소리
자막: 필수
BGM: 잔잔한 피아노 / 클래식

이미지 스타일: 따뜻한 일러스트, 우나어 브랜드 컬러(#FF6F61)

출력:
  - 16:9 (유튜브 / 구글 애즈)
  - 9:16 (인스타그램 릴스 / 유튜브 숏)

저장: assets/generated/video/[오늘날짜]_[주제]/"
```

```
CTR, CPC, 클릭수, 노출수 요약하고
기준치(CTR 1%, CPC 300원) 미달 캠페인 있으면 알려줘."
```

## C-Level 에이전트 요청 예시

### CMO → Graphic Designer
```
CMO가 이번 주 Threads 콘텐츠에 이미지 필요:
"이번 주 Threads 포스팅 5개에 맞는 이미지 만들어줘.
주제: [knowledge-base.ts의 이번 주 전략]
브랜드 톤: 우나어, #FF6F61, 따뜻함
사이즈: 1080×1080
저장: assets/generated/sns/[날짜]_threads/"
```

### CPO → Product Designer
```
CPO가 여정 분석 후 개선 화면 필요:
"마이페이지 이탈 포인트 개선 화면 Figma에 설계해줘.
문제: [journey-analyzer.ts 데이터 기반]
개선 방향: [구체적 내용]
데스크탑 + 모바일, 승인 후 코딩 지시 예정"
```
