## 5 Whys — 욕망의 뿌리

### "왜 인생설계 글에 조회수 9,253이 나오는가?"
1. Why? → 실버타운·주거 고민이 많다
2. Why? → 노후를 어디서 어떻게 살지 모른다
3. Why? → 퇴직 후 삶의 지도가 없다
4. Why? → 인생 다음 챕터에 대한 사회적 준비·문화·대화가 없다
5. **핵심**: "이 나이에 어떻게 살아야 하는지 아무도 안 알려줬다"

### "왜 건강 글은 많이 쓰지만 반응이 낮은가?"
1. Why? → 건강 정보는 공감보다 검색 목적
2. Why? → 갱년기·증상은 '나만의 비밀'처럼 느껴짐
3. Why? → 체면 장벽 — 아프다고 드러내기 어려움
4. Why? → 건강 불안의 본질은 "내가 혼자 이걸 감당해야 하나?"
5. **핵심**: 건강 뒤에는 고립감(RELATION)이 있다

### "왜 외로움 키워드는 글 수가 적은데 반복 등장하는가?"
1. Why? → 외로움을 직접 표현하면 체면 손상
2. Why? → 외로움은 건강/유머/일상 글 속에 녹아있다
3. Why? → "혼자다"라고 말하는 게 아니라 글을 쓰러 온다
4. Why? → 커뮤니티 자체가 외로움 해소 행동
5. **핵심**: 우나어에 오는 행위 자체가 RELATION 욕망의 표현

## 확정된 욕망 순위

```
#1 RELATION: 외로움·느슨한 연결
   → 체면 뒤에 숨은 가장 깊은 욕망
   → 커뮤니티에 오는 행위 자체가 이 욕망의 표현

#2 RETIRE: 인생의 의미·다음 챕터
   → 퇴사·주거·인생2막 반응 폭발 (조회수 최다)
   → "어떻게 살아야 하는지 모른다"

#3 MONEY: 연금·노후자금·재무 설계
   → 재무 불안, 연금 키워드 3위
   → 글 수는 적지만 반응 강도 높음

#4 HEALTH: 건강불안·갱년기
   → 글 수 가장 많음 (정보 갈증)
   → 공감보다 정보 니즈, 건강 불안 뒤엔 RELATION이 있음
```
