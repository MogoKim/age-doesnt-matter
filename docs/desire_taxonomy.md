# 욕망 분류 체계 (Desire Taxonomy)

우나어 크롤러 AI 분석 및 에이전트 인텔리전스의 공통 기준.
50-60대 커뮤니티 게시글에서 심리/욕망/니즈를 추출할 때 이 분류를 사용한다.

---

## 1. 4축 분류 (Desire Type)

| 코드 | 이름 | 정의 | 예시 신호 |
|------|------|------|----------|
| `big_desire` | 큰 욕망 | 삶에서 궁극적으로 원하는 상태 | "건강하게 오래 살고 싶다", "손주들 곁에 있고 싶다", "의미 있는 하루를 보내고 싶다" |
| `need` | 니즈 | 큰 욕망을 위해 지금 필요한 조건 | "병원 정보가 필요하다", "일자리가 필요하다", "이 증상이 뭔지 알고 싶다" |
| `want` | 원츠 | 선호하는 방식/수단/형태 | "복잡하지 않게", "믿을 수 있는 사람에게서", "빠르게 알 수 있으면" |
| `demerit` | 디메리트 | 현재의 마찰/불만/장벽 | "광고가 너무 많다", "설명이 어렵다", "아무도 대답을 안 해준다" |

---

## 2. 욕망 카테고리 (Desire Category) — 13개

| 코드 | 카테고리 | 대표 주제 | 대표 신호 표현 |
|------|----------|----------|------------|
| `HEALTH` | 건강/증상/병원 | 갱년기, 관절, 당뇨, 혈압, 약, 병원, 수술, 영양제 | "병원 어디 가요", "이 증상이", "약 먹어야 하나" |
| `FAMILY` | 가족/자녀/남편/손주 | 자식 걱정, 남편 갈등, 손주 자랑, 시댁 | "자식이", "남편이", "손주", "며느리가" |
| `MONEY` | 돈/재테크/생활비 | 연금, 투자, 절약, 보험, 노후 자금, 부동산 | "연금", "투자", "절약", "노후 자금", "어디 넣어야" |
| `RETIRE` | 은퇴/노후/인생2막 | 퇴직, 할 일, 의미, 우울, 새 시작, 은퇴 계획 | "퇴직하고", "은퇴 후", "뭘 해야 할지", "허전하다" |
| `JOB` | 일자리/자격증/부업 | 알바, 파트타임, 자격증, 재취업, 부업, 수입 | "알바", "자격증", "재취업", "뭔가 해야 할 것 같다" |
| `RELATION` | 관계/외로움/소통 | 친구 없다, 서럽다, 혼자다, 대화 상대, 공감 | "친구가 없다", "외롭다", "말 걸 곳이", "이해해주는 사람" |
| `HOBBY` | 취미/여가/활동 (직접 참여) | 등산, 여행, 배움, 시간 활용, 귀농귀촌 | "등산", "여행", "텃밭", "뭔가 배워보고 싶다" |
| `MEANING` | 삶의 의미/철학/감사 | 이 나이에, 살아있다, 감사하다, 왜 사나, 보람 | "이 나이에", "살아있다는 게", "보람", "감사하다" |
| `DIGNITY` | 존중/인정/자존감 | 무시당하는 느낌, 고부갈등, 나이 차별, 가치 인정 | "무시당한다", "취급이 달라", "아직 할 수 있는데", "나이 차별" |
| `LEGACY` | 자식에게 남기기/기억 | 내가 죽으면, 뭔가 해주고 싶다, 기억되고 싶다 | "내가 죽으면", "자식들한테 해주고 싶다", "뭔가 남겨야", "기억되고 싶다" |
| `CARE` | 돌봄/간병/의존 | 부모 간병, 배우자 돌봄, 내가 아프면 누가 봐줄까 | "부모님 간병", "남편을 돌봐야", "내가 아프면 누가", "요양원" |
| `FREEDOM` | 자유/독립/나만의 시간 | 가족·의무에서 벗어나 나만의 공간과 시간 | "혼자 있고 싶다", "나만의 시간", "벗어나고 싶다", "나를 위한" |
| `ENTERTAIN` | 연예/드라마/팬덤/즐거움 (소비형) | 드라마, 트로트, 연예인 팬덤, 콘서트 | "임영웅", "트로트", "드라마 결말", "콘서트", "배우가 너무 좋아" |

> **HOBBY vs ENTERTAIN**: HOBBY는 내가 직접 몸을 움직여 하는 활동. ENTERTAIN은 콘텐츠·연예인을 팬으로서 보고 즐기는 것.
> **CARE vs FAMILY**: FAMILY는 감정적 관계(사랑, 갈등, 그리움). CARE는 돌봄 부담과 의존 공포.
> **FREEDOM**: 50-60대 여성 특유. 기존 카테고리에 잘 포착되지 않던 욕망.
> **DIGNITY**: 나이 들수록 강해지는 "나는 아직 가치 있는 존재" 욕구. 고부갈등/세대 갈등에서 자주 나타남.

## 3. 심리 상태 (Emotion Tags) — 10가지

| 코드 | 심리 | 신호 표현 | 반응 방향 |
|------|------|----------|----------|
| `LONELY` | 외로움 | "혼자다", "아무도 모른다", "이해해주는 사람 없다", "대화 상대가 없다" | 공감, 연결, 소속감 |
| `ANXIOUS` | 불안 | "무섭다", "걱정이다", "어떡하나", "앞이 안 보인다", "이 증상이 뭐지" | 정보 제공, 안심, 검증 |
| `ANGRY` | 분노/억울 | "화난다", "억울하다", "짜증난다", "왜 나만", "이해할 수 없다" | 공감, 편 들어주기 |
| `HOPEFUL` | 기대/희망 | "이제라도", "혹시", "해보고 싶다", "기대된다", "도전해볼까" | 응원, 정보 지원 |
| `RESIGNED` | 체념/포기 | "이 나이에", "어쩌겠어", "그냥 살지", "이제 다 됐어", "포기" | 동기부여, 재프레이밍 |
| `GRATEFUL` | 감사/위로 | "그래도", "감사하다", "괜찮아진 것 같다", "위로받았다" | 공감 강화, 연대 |
| `PROUD` | 자랑/성취 | "해냈다", "잘됐다", "보람있다", "뿌듯하다", "성공했다" | 축하, 인정 |
| `JEALOUS` | 부러움/박탈감 | "저 사람은 되는데 나는", "상대적으로", "부럽다", "왜 나는 안 되나" | 현실적 공감, 가능성 제시 |
| `CONFUSED` | 혼란/정보 과부하 | "뭘 믿어야 할지", "너무 많다", "모르겠다", "헷갈린다" | 정리, 단순화, 신뢰할 수 있는 정보 |
| `NOSTALGIC` | 그리움/향수 | "그땐 좋았는데", "예전에는", "돌아가고 싶다", "그리워", "옛날 생각" | 추억 공감, 현재와 연결 |

> 감정 태그는 복수 가능 (한 글에 최대 3개). 감정이 명확하지 않으면 [] 반환 — 억지로 붙이지 않는다.
> `NOSTALGIC`: 50-60대 커뮤니티 특유의 감정. "예전에는", "그때는", "그리워" 신호로 포착.

## 4. 긴급도 (Urgency Level) — 1-5

| 점수 | 수준 | 신호 | 예시 |
|------|------|------|------|
| `5` | 즉각적 고통/위기 | 지금 당장 | "오늘 병원에 가야 하는데", "당장 돈이 없다", "이 약 먹어도 되나" |
| `4` | 근 미래 불안 | 이번 주~이번 달 | "다음 달이 걱정", "이번 주에 결정해야", "이 증상이 점점 심해진다" |
| `3` | 중기 고민 | 올해 안에 | "올해 안에 해야 할 것 같다", "언젠가는 해야 할 것 같아서" |
| `2` | 막연한 걱정 | 언젠가 | "나중에 어떡하지", "그냥 불안해", "막막하긴 한데" |
| `1` | 일상적 관심 | 정보 공유/수다 | 정보 공유, 취미 이야기, 일상 수다, 자랑 |

## 5. 핵심 신호 표현 사전

### 욕망/갈등 신호 (Desire/Conflict Signals)
```
"힘들다", "불안하다", "짜증난다", "서럽다", "억울하다"
"이제라도", "그래도", "혹시", "왜 나는", "어떡하나"
"모르겠다", "걱정이다", "무섭다", "두렵다"
```

### 원츠/디메리트 신호 (Want/Demerit Signals)
```
"편했으면", "빨랐으면", "덜 귀찮았으면", "쉬웠으면"
"광고가 너무", "설명이 어렵다", "아무도 안 알려준다"
"이런 게 있으면 좋겠다", "왜 없지", "아쉽다"
```

### 긴급 신호 (Urgency Signals)
```
"오늘", "지금", "당장", "이번 주", "급해요"
"어떡해야 하나요", "빨리", "갑자기", "갑작스럽게"
```

## 6. communitySignal (글을 쓴 이유)

| 코드 | 의미 | 신호 |
|------|------|------|
| `question` | 무언가 물어보러 왔다 (정보 갈증) | "어떻게 해요?", "알려주세요", "추천해 주세요" |
| `complaint` | 하소연/불만 배출 | "너무 힘들어요", "화가 나서요", "답답해서", "들어줘요" |
| `confession` | 고백/털어놓기 | "사실은...", "아무도 몰랐는데", "부끄럽지만", "이런 말 처음 하는데" |
| `recommendation` | 경험 공유/추천 | "이거 해봤는데", "좋았어요", "추천해요", "꼭 해보세요" |
| `celebration` | 자랑/기쁨 나누기 | "드디어!", "해냈어요!", "너무 기쁜 일이", "자랑하고 싶어서" |

## 7. ageSignal (추론 연령대)

| 코드 | 추론 기준 |
|------|---------|
| `50s` | 직장 언급, 자녀 중고등·대학, "요즘 50대가", 갱년기 초기 |
| `60s` | 은퇴 언급, 자녀 취업·결혼, 손주, "환갑", 연금 수령 |
| `70s+` | "칠순", "오래 살았네", 배우자 사별, 중증 건강 이슈, 요양 |
| `unknown` | 연령 신호 없음 |

## 8. 엔터테인먼트 감지 신호 (ENTERTAIN 카테고리)

아래 키워드가 제목/본문에 포함 시 `desireCategory: 'ENTERTAIN'` 우선 고려:
```
연예인 이름, 드라마 제목, 트로트, 임영웅, 송가인, 나훈아, 영탁,
넷플릭스, 유튜브, 콘서트, 팬미팅, 배우, 가수, 응원봉,
드라마 결말, 시청률, OST, 뮤지컬, 유명인, 아이돌
```

> ENTERTAIN이 전체 수집 글의 10% 이상이면 엔터 전담 페르소나(EN1~EN5) 활성화.

## 9. 분석 원칙

1. **글 + 댓글 같이 본다**: 제목 < 본문 < 댓글 순으로 진짜 감정이 강해짐
2. **정보글보다 감정글 우선**: 건강 정보 공유보다 "이 증상 나만 이렇나요?"가 욕망을 더 잘 보여줌
3. **desireType은 하나만**: big_desire / need / want / demerit 중 가장 지배적인 것 하나 선택
4. **emotionTags는 복수 가능**: 한 글에 여러 감정이 섞일 수 있음 (최대 3개)
5. **desireCategory도 하나**: 가장 지배적인 카테고리 하나 선택
6. **urgencyLevel은 보수적으로**: 확실하지 않으면 낮게 (1-2) 설정
7. **communitySignal + ageSignal**: 글의 의도와 연령대를 추론 — 없으면 unknown/null 허용

## 10. AI 분석 시 참고 예시

### 예시 1 — HIGH urgency, 복합 욕망
```
제목: "갱년기 증상인지 모르겠어요 가슴이 너무 두근두근해서"
본문: "요즘 갑자기 심장이 막 두근거리고 얼굴이 화끈거려요. 병원을 가야 하는 건지..."
댓글: "저도 그랬어요! 갱년기 시작인 것 같아요", "빨리 병원 가보세요"

분석:
- emotionTags: ["ANXIOUS"]
- desireCategory: "HEALTH"
- desireType: "need" (병원 정보, 진단 필요)
- urgencyLevel: 4
- psychInsight: "갑작스러운 신체 증상에 대한 불안, 갱년기 여부 확인 필요"
```

### 예시 2 — LONELY, MEANING
```
제목: "퇴직하고 나서 하루가 너무 길어요"
본문: "남편은 아직 출근하고 저 혼자 집에 있으니까 뭘 해야 할지 모르겠어요. 친구들은 다들 바빠서..."
댓글: "저도 그랬어요. 시간이 지나면 적응돼요", "취미를 찾아보세요"

분석:
- emotionTags: ["LONELY", "RESIGNED"]
- desireCategory: "MEANING"
- desireType: "big_desire" (의미 있는 하루, 소속감)
- urgencyLevel: 3
- psychInsight: "은퇴 후 정체성 공백과 외로움, 새 루틴/커뮤니티 필요"
```

### 예시 3 — MONEY, ANXIOUS, HIGH urgency
```
제목: "국민연금 납부 중단하면 어떻게 되나요?"
본문: "형편이 어려워서 이번 달부터 못 낼 것 같아요. 나중에 문제 생기나요?"

분석:
- emotionTags: ["ANXIOUS"]
- desireCategory: "MONEY"
- desireType: "need" (연금 정책 정보 필요)
- urgencyLevel: 4
- psychInsight: "재정 압박으로 인한 연금 포기 고려, 장기 노후 불안"
```
