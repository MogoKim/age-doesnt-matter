2-5. 품질 점수 공식

품질점수 = (참여도 × 40%) + (본문길이 × 20%) + (미디어 × 5%) + (게시판우선도 × 20%) + (최신성 × 15%) 30점 미만 → 저장 안 함 (버림) 30~59점 → DB 저장, isUsable=false (참고용만) 60점 이상 → isUsable=true (매거진·트렌드 분석에 활용)

Section 03

트렌드 분석 — 수집된 글에서 무엇을 파악하는가

3-1. 1단계: 심리 분석 (Claude Haiku, 5개씩 배치)

감정 태그 (최대 3개)
  • ANXIOUS — 불안
  • LONELY — 외로움
  • ANGRY — 분노
  • HOPEFUL — 희망
  • RESIGNED — 체념
  • GRATEFUL — 감사
  • PROUD — 자랑
  • JEALOUS — 상대적박탈감
  • CONFUSED — 혼란
  • NOSTALGIC — 그리움
욕망 카테고리 (1개 선택)
  • HEALTH · FAMILY · MONEY · RETIRE · JOB
  • RELATION · HOBBY · MEANING · DIGNITY
  • LEGACY · CARE · FREEDOM · ENTERTAIN

  • 긴급도 1~5점
  • 5=즉각위기 / 4=근미래불안
  • 3=중기고민 / 2=막연한걱정
  • 1=일상관심
🧬
갈등 DNA (긴급도 3이상 하소연 글에만): viralType — BETRAYAL(배신) / INJUSTICE(부당함) / CONTROVERSY(논쟁) / REVERSAL(반전) / EMPATHY(공감) · commentSplit — 0~10점 (10=찬반극단, 0=전원공감)

3-2. 2단계: 트렌드 종합 (Claude Sonnet → CafeTrend DB)

DB 필드내용
hotTopics오늘의 핫 주제 7개 (건강·가족·돈·취미·관계·유머·의미 영역별)
keywords자주 등장한 단어 상위 15개
desireMap13개 욕망 카테고리별 오늘 비율 (%)
emotionDistribution10개 감정별 오늘 비율 (%)
magazineTopics오늘 발행 추천 주제 3개 (점수 포함)
controversyTopics논쟁 가능성 높은 주제 최대 2개

3-3. 3단계: 욕망 지도 → 페르소나 할당량 계산

페르소나 욕망 친화도 × 오늘 욕망 비율 = 활동 배율 (0.5배 ~ 2.0배) 예: 오늘 HEALTH 욕망이 25%로 높으면 → 건강 페르소나 (걷기매니아·근육할머니·약국단골 등)가 오늘 더 많이 활동
