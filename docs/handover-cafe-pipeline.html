댓글 생성 단계 — 상세

1

게시글 인풋 준비

게시글 제목 전체 + 본문 앞 300자만 사용. 전체 본문은 비용·토큰 절약을 위해 truncate.

2

카페 댓글 예시 주입 (말투 모방)

실제 카페에서 수집한 댓글 2~3개를 System Prompt에 삽입. HTML 태그·엔티티 전처리 후 220자 발췌. 모델이 길이·톤·어미를 자연스럽게 모방하도록 유도. 내용 인용은 절대 금지, 스타일만 참고.

3

최근 댓글 이력 조회 (반복 방지)

해당 페르소나가 최근 단 댓글 3개 조회 → 첫 문장·표현이 겹치지 않도록 System Prompt에 추가.

4

트렌드 말투 주입 (DailyBrief에서)

오늘의 DailyBrief 중 어휘·말투 스타일을 추가 주입. 현재 커뮤니티에서 유행하는 표현을 반영해 동시대감 부여.

5

생성 후 필터링

마크다운 기호 전면 제거 (stripMarkdown). 줄바꿈 제거. 최종 1~3문장 짧은 공감·경험 공유 형태.

ℹ️
SHEET 게시글 차단: SheetViral 게시글(source='SHEET')에는 일반 시드봇 댓글이 달리지 않음. 4파동 전용 봇이 별도로 담당.
