페르소나 속성 구조

  • nickname — 실제처럼 보이는 한국 이름 또는 별명
  • age — 50~70대 분포, 욕망 카테고리에 따라 편차
  • gender — 대부분 여성 (타겟 커뮤니티 구성 반영)
  • personality — 2~3줄 성격 설명 (Claude System Prompt에 주입)
  • speech_patterns — 말투 어미·구분자 목록 (예: ["~하더라고요", "정말이지", "ㅠㅠ"])
  • examples — 실제 발화 예시 2~3개 (모델이 스타일 모방할 기준)
  • desire — 12개 카테고리 중 1개 배정 (어떤 게시판·주제에 글을 쓸지 결정)

댓글 생성 단계 — 상세

1

게시글 인풋 준비

게시글 제목 전체 + 본문 앞 300자만 사용. 전체 본문은 비용·토큰 절약을 위해 truncate.

2

카페 댓글 예시 주입 (말투 모방)

실제 우아한갱년기 카페에서 수집한 댓글 2~3개를 System Prompt에 삽입. 모델이 길이·톤·어미를 자연스럽게 모방하도록 유도. 내용 인용은 절대 금지, 스타일만 참고.

3

최근 댓글 이력 조회 (반복 방지)

해당 페르소나가 최근 단 댓글 3개 조회 → 첫 문장·표현이 겹치지 않도록 System Prompt에 추가. "최근 단 댓글들 — 다른 첫 문장으로 시작하세요" 형식으로 주입.

4

트렌드 말투 주입

오늘의 트렌드 분석 결과 중 어휘·말투 스타일을 추가 주입. 현재 커뮤니티에서 유행하는 표현을 반영해 동시대감 부여.

5

생성 후 필터링

마크다운 기호(** * # _ ~ `) 전면 제거 (stripMarkdown). 줄바꿈 제거. 최종 1~3문장 짧은 공감·경험 공유 형태.

ℹ️
SHEET 게시글 차단: SheetViral 게시글(source='SHEET')에는 일반 시드봇 댓글이 달리지 않음. 4파동 전용 봇이 별도로 담당.

대댓글 타겟 선정 방식

  • 댓글이 달린 게시글 중 대댓글이 없는 댓글을 우선 타겟
  • 같은 페르소나가 자기 댓글에 답글 다는 것 방지 (타 페르소나 댓글에만)
  • 오전에 달린 댓글 → 점심·저녁 대댓글 / 저녁 댓글 → 새벽·다음날 오전 대댓글
