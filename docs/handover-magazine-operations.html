왜 매거진인가 — 목적과 배경

📌 댓글 정책 (2026-06-04~)
매거진(MAGAZINE)은 SEO·정보 제공용 콘텐츠다. 봇 댓글·대댓글·좋아요(wave 포함)는 운영하지 않는다. 사람 댓글(회원·비회원)만 허용한다.
구현: agents/core/bot-engagement-policy.ts의 isBotEngagementEnabledBoard()가 STORY/HUMOR/LIFE2만 허용하고, 모든 봇 engagement 생성 직전(comment-activator / reply-chain-driver / controversy-chain / wave-processor / seed·micro-scheduler)에서 MAGAZINE을 차단한다. 레거시 commentWaveQueue가 남아도 생성 직전 guard로 막힌다(skip 사유: board_engagement_disabled).
우리 서비스에 오는 50~60대 여성을 위해 매일 신선한 정보를 자동으로 생산하는 시스템.
신뢰 구축 → 재방문 유도 → 회원 전환 → 광고 수익의 4단계 엔진이다.
🤔
문제
카페/커뮤니티만으로는 신규 방문자가 "이 사이트를 믿어도 되나?" 판단할 근거가 없다. 처음 온 사람에게 보여줄 정보성 콘텐츠가 필요하다.
📰
해결책
AI가 매일 1,500~2,000자 정보형 기사를 자동 발행. 50~60대가 검색하는 건강, 재테크, 은퇴 주제를 다룬다. 창업자 개입 없이 돌아간다.
🎯
목표
검색 유입 → 기사 읽기 → 댓글/좋아요 → 회원 가입. 쌓인 기사가 SEO 자산이 되어 복리처럼 트래픽을 늘려간다.

매거진은 우나어를 "50~60대가 믿을 수 있는 정보 허브"로 포지셔닝하는 핵심 콘텐츠다. 커뮤니티 기능만 있으면 신규 방문자가 이탈하지만, 양질의 기사가 쌓이면 "이 사이트에는 뭔가 있다"는 신뢰가 생긴다. 그 신뢰가 회원 가입과 재방문으로 이어진다.

AI는 어떤 주제를 어떻게 고르는가

주제 선정 우선순위 4단계

1
시리즈 편 (월요일만) 연간 16개 시리즈(편당 5편, 총 80편) 계획에서 오늘 차례가 된 편. 예: "50대 갱년기 완벽 가이드 3편". 가장 높은 우선순위 — 시리즈는 독자 재방문을 만드는 핵심.
2
GEO 시드 (지역 기반 주제) 구글 GEO 랭킹을 위해 설계된 지역 특화 주제. "서울 50대 여성 노후 준비 현실" 같은 식. 구글 검색 유입에 특히 효과적.
3
카페 트렌드 주제 어제/그제 우리 카페에서 많이 언급된 주제. 실제 50~60대가 지금 궁금해하는 것 → 공감도 높은 기사.
4
욕망 폴백 (최후 수단) 위 3가지가 없을 때. 건강·재테크·은퇴준비 중 오늘 덜 다룬 주제를 자동 선택. 기사가 0편으로 끝나지 않기 위한 안전망.

중복 방지 필터

최근 7일간 발행된 기사 제목과 비교해 제목 첫 3단어 중 2개 이상 겹치면 자동 제외한다(getRecentMagazineTitles(7) + isSimilarTitle()). "50대 무릎 통증"에 대한 글을 3일 전에 썼다면 오늘은 다른 주제로 넘어간다. 추가로 전체 기간 제목 정규화 완전중복 가드(isDuplicateAllTime(), 8cbcb16)가 있어 과거에 똑같은 제목이 있으면 영구 차단한다. 독자가 "또 같은 내용이네"라고 느끼지 않도록 하는 장치다.

AI는 어떻게 글을 쓰는가 — 페르소나와 톤

AI가 "정순씨 에디터봇"으로 쓴다 기사는 AI가 쓰지만, 독자에게는 "우나어 에디터팀이 작성한 기사"로 보인다. AI는 항상 50~60대 여성 독자의 시선에서, 공감 먼저 정보 나중 순서로 쓴다.

5개 검증 페르소나 — AI가 항상 이 사람들을 머릿속에 두고 쓴다

P2 정희씨건강 염려, 악화 공포"이 증상 내가 어떻게 해야 해?" 구체적 답

SEO를 위한 롱테일 키워드 전략

AI는 기사마다 해당 카테고리의 롱테일 키워드를 제목이나 본문에 자연스럽게 녹인다. "건강"이라는 단어보다 "50대 여성 밤에 식은땀 나는 이유"처럼 구체적인 검색어를 타겟한다. 이런 롱테일 키워드는 경쟁이 적어 신생 도메인도 상위 노출이 가능하다.

건강 예시: "50대 여성 밤에 식은땀 나는 이유", "갱년기 수면 장애 해결법", "무릎 통증 운동하면 낫나요"
재테크 예시: "퇴직금 IRP 넣으면 세금 얼마나 아끼나", "국민연금 조기수령 vs 65세 비교"
은퇴준비 예시: "퇴직 후 첫 한 달 어떻게 보내야 하나요", "남편 퇴직 후 24시간"

일요일에 Opus를 쓰는 이유

평일(월~토)은 Claude Sonnet(중간 모델)으로 빠르고 효율적으로 기사를 생산한다. 하지만 일요일은 Claude Opus(최상위 모델)로 전환한다. 일요일 독자는 더 여유롭게 긴 글을 읽고, 주간 정리·전략적 주제 기사가 적합하기 때문이다. 비용은 더 들지만 품질 차이가 확연하다.

이미지는 어떻게 만들어지는가

모든 기사에는 히어로 이미지가 반드시 필요하다. 이미지 없는 기사는 발행 자체가 보류된다. AI가 기사 내용에서 핵심 장면을 추출해 영어 설명으로 변환 → ChatGPT(Playwright)가 이미지 생성. (2026-06-04 Gemini 구독 종료로 ChatGPT 전환, API 미사용 = 추가 비용 0)

이미지 생성 우선순위

1순위
ChatGPT AI 생성 (chatgpt.com Playwright) 기사 내용 기반 영문 프롬프트로 chatgpt.com에 접속해 이미지 생성. 한국 여성 4050 실사 또는 음식·풍경 사진. 성공 시 R2에 업로드(magazine-chatgpt-*.png). ChatGPT Pro 구독 세션 사용.
2순위 폴백
Unsplash 사진 라이브러리 (인물 제외) ChatGPT 실패 시 FOOD·SCENE·OBJECT만 Unsplash API에서 관련 사진 자동 검색+다운로드. PERSON_REAL(인물)은 외국 스톡 방지를 위해 폴백 금지 → 발행 보류로 넘어감.
발행 보류
인물 이미지 실패 → 기사 발행 중단 히어로 이미지 없이는 발행하지 않는다. 다음 세션 때 같은 주제로 재시도하거나 다른 주제로 교체.

ILLUSTRATION개념·차트형 주제현재 비활성 — DALL-E 꺼둠
ChatGPT 세션이 만료되면 ChatGPT는 크롬 브라우저 자동화로 이미지를 생성한다(프로필 ~/.chrome-chatgpt-profile). 로그인 여부는 /api/auth/session으로 정확히 감지하며, 미로그인 상태로 자동(launchd) 실행되면 무한 대기 대신 즉시 실패하고 Slack 알림이 온다. 보통 수주~수개월에 한 번 발생. 알림이 오면 chatgpt.com에 다시 로그인하면 된다.
SingletonLock 근본 해결 (2026-06-04) 과거 만성 장애의 진짜 원인은 "이미지 생성용 크롬을 열고 종료하지 않아" 고아 프로세스가 프로필 락(SingletonLock)을 점유 → 다음 실행이 전부 실패한 것이었다. 이제 매 실행 종료 시 브라우저를 반드시 닫아(모든 종료 경로) 락 누적을 차단한다.

카페 데이터와의 연계 — 실제 독자가 궁금한 것을 쓴다

매거진은 창작이 아니다. 우리 카페에서 실제 50~60대가 오늘 무슨 이야기를 하고 있는가를 보고 그것을 기사로 만든다. 이것이 다른 AI 콘텐츠와의 결정적 차이다.

카페 → 매거진 데이터 흐름
1
카페 크롤러가 매일 네이버 카페 2곳에서 인기 글을 자동 수집
2
트렌드 분석 에이전트가 "오늘 가장 많이 언급된 주제"를 추출
3
매거진 생성 시 해당 트렌드 주제를 3순위 후보로 활용
4
기사 작성 시 카페 원글(최대 5개)을 AI에게 참고자료로 제공
5
기사 본문에 "우나어 커뮤니티에서도 자주 나오는 주제예요" 형태로 자연 언급
왜 이것이 중요한가 AI가 일반적인 건강 정보를 쓰는 것과, "지금 우리 독자들이 실제로 고민하는 것"을 쓰는 것은 공감도가 다르다. 카페 데이터 연계가 매거진을 "내 이야기 같은 기사"로 만드는 비결이다.

수익화 구조 — 매거진이 돈을 버는 방법

수익원위치현황
Google AdSense 기사 상세 페이지 내 ✅ 활성 — 트래픽 증가 시 수익 증가
쿠팡 배너 매거진 목록 페이지 (8번째 카드 뒤) ✅ 활성
쿠팡 CPS 링크 기사 하단 관련 상품 추천 ⚠️ 프론트 비활성 — CPS_ENABLED=false (magazine/[id]/page.tsx). 발행 시 CPS 링크는 DB에 저장되나 화면 렌더는 꺼져 있음(상품 준비 후 활성화 예정). Coupang API 401도 발행에는 무영향(원본 URL 폴백).
SEO 자연유입 구글 검색 → 기사 → 사이트 누적 중 — 장기 자산
매거진의 수익은 직접 수익보다 간접 수익이 크다 AdSense + 쿠팡 배너 수익보다, 매거진이 만드는 "신뢰 → 회원 가입 → 커뮤니티 활성화 → 광고 단가 상승" 연쇄가 더 가치 있다. 매거진은 수익 센터가 아니라 신뢰 제조 엔진이다.

창업자가 알아야 할 3가지

딱 이것만 알면 된다

1
매거진은 완전 자동 — 창업자가 할 일 없다 네이버 블로그와 달리, 매거진은 창업자가 아무것도 안 해도 매일 기사가 발행된다. Slack #매거진 알림이 오면 "오늘도 잘 돌아가고 있구나" 확인만 하면 된다. 알림이 없는 날만 "왜 없지?" 확인.
2
Mac이 켜져 있어야 한다 매거진 시스템은 창업자 Mac에서 실행된다. Mac이 꺼져 있으면 12:00와 14:00에 아무 일도 일어나지 않는다. 절전 모드는 괜찮지만 완전 종료하면 안 된다. 장기 외출 시 이 점을 반드시 체크.
3
효과는 천천히, 하지만 복리다 오늘 발행한 기사가 내일 당장 트래픽을 만들지 않는다. 구글 색인에 1~3일, 검색 상위에는 수개월 걸린다. 하지만 한 번 쌓인 기사는 계속 사람들을 데려온다. 6개월 후의 트래픽은 지금 매일 발행하는 기사들의 합산이다.
