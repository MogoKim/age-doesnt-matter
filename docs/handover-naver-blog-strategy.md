# 네이버 블로그 운영 기획서 — 우리 나이가 어때서

> 작성일: 2026-05-14 | 담당: 창업자 직접 운영 (반자동 파이프라인)

---

## 1. 운영 목적

| 목적 | 내용 |
|------|------|
| 트래픽 유입 | 네이버 블로그 → age-doesnt-matter.com 유입 확대 |
| SEO | 네이버 C-Rank + AI 브리핑 노출로 자연 검색 유입 확보 |
| 브랜드 인지 | 50대·60대 여성에게 "우리 나이가 어때서" 브랜드 각인 |
| 체험단/협찬 | 블로그 팔로워 누적 후 제품 체험단 신청 기반 마련 |

## 3. 발행 파이프라인 (반자동)

```
launchd 자동 실행 (12:30 / 18:30 KST)
        ↓
poster.ts 실행
  1. DB에서 오늘 매거진 포스트 조회
  2. LLM → 네이버 SEO 최적화 변환 (blogTitle + sections + hashtags)
  3. 이미지 수집: 썸네일 + 본문 이미지 + Gemini 생성 (총 6개 보장)
  4. NaverBlogQueue DB 저장 (status: READY_FOR_MANUAL)
  5. Slack 알림 → "📝 Naver 발행 대기"
        ↓
창업자 /admin/naver-blog 접속
  - 제목 복사 → 네이버 블로그에 붙여넣기
  - 본문 복사 (섹션별 or 전체)
  - 해시태그 복사
  - 이미지 6개 저장 후 업로드
  - 발행 완료 후 [발행 완료] 클릭 → 큐에서 삭제
```

### 실행 환경

| 항목 | 내용 |
|------|------|
| 자동 실행 | launchd (로컬 Mac) |
| 아침 발행 | `com.unaeo.naver-blog-morning.plist` → UTC 03:30 (KST 12:30) |
| 저녁 발행 | `com.unaeo.naver-blog-evening.plist` → UTC 09:30 (KST 18:30) |
| 어드민 URL | `/admin/naver-blog` |

## 4. 발행 정책

| 항목 | 기준 |
|------|------|
| 일 최대 발행 수 | 2건/일 (안정화 30일간: 1건/일) |
| 1회 실행 | catch-up 1건 + 정기 1건 = 최대 2건 |
| PENDING 만료 | 48시간 초과 → 자동 EXPIRED |
| HALT 조건 | FAILED 2건 이상 → 전면 중단 |

## 5. 콘텐츠 기준 (SEO)

| 항목 | 기준 |
|------|------|
| 목표 글자수 | 1,800자 (최소 1,000 / 최대 6,000) |

## 6. 띠배너 운영 전략

블로그 글 내에 우나어 홍보 배너 삽입.

### 삽입 위치

| 위치 | 목적 |
|------|------|
| 글 중간 | 읽다가 자연스럽게 클릭 유도 |
| 글 마지막 | 다 읽은 독자에게 CTA |

### 배너 규격

- 크기: **730 × 180px** (네이버 블로그 본문 최적)
- 배경: 다크(#1A0A0A) 또는 코랄 그라데이션
- 핵심 문구 후보 (5안):

| 안 | 문구 | 타입 |
|----|------|------|
| 1 | 나만 이런 건 줄 알았는데… | 공감형 |
| 2 | 우리 또래, 솔직하게 얘기해요 | 커뮤니티형 |
| 3 | 50대부터 제대로 사는 법 | 정보형 |
| 4 | 지금 이 나이, 나쁘지 않아요 | 위로형 |
| 5 | 혼자 고민하지 말아요 | 공감형 |

## 7. 유입 추적 방법

### 현재 방식 (referrer 기반)
- 네이버 블로그 링크 클릭 시 브라우저가 자동으로 `referrer` 전송
- EventLog 테이블에 저장 → DB 조회로 확인 가능
- **포스팅별 클릭 구분 가능** (referrer URL에 포스팅 번호 포함)

### 확인 방법
```bash
# 특정 기간 naver 유입 조회
npx tsx --env-file=.env.local _naver-check.mts
```

### 실적 (2026-05-13~14, 발행 이틀 차)
| 출처 | 건수 |
|------|------|
| 네이버 블로그 포스팅 클릭 | 4건 |
| 네이버 블로그 메인 | 4건 |
| 네이버 검색 직접 유입 | 2건 |
| bit.ly 단축 URL | 1건 |
| **실제 외부 유입 합계** | **11건** |

> 네이버 PV는 블로그 자체 통계로 확인. 우나어 DB는 실제 유입 클릭 추적용.
