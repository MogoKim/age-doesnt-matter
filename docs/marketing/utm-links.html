A-1. 앱 유입용 안드=앱 / iOS·PC=웹

A-2. 웹 유입용 무조건 웹 홈

⚠️ 같은 채널이라도 A-1(/go)과 A-2(/?utm)를 동시에 쓰지 마세요. 앱 깔리길 원하면 /go, 웹으로 바로 보낼 거면(구글애즈 등) /?utm.
