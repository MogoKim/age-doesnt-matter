## A3. 네비게이션 시스템

> **핵심 원칙**: 하단 탭 바 없음. 상단 네비게이션으로 통일.
> 사용자가 글쓸 수 있는 페이지에서는 플로팅 글쓰기 버튼(FAB)으로 유도.

### A3-1. 모바일 상단바 (Sticky)

```
┌──────────────────────────────┐
│  [🟠Logo]  우나어     🔍  👤  │ ← 상단바: height 56px, sticky
├──────────────────────────────┤
│  ⭐     💼     💬     ⚡    📖 │ ← 아이콘 메뉴: height 64px
│ 베스트 내일찾기 사는이야기 활력충전소 매거진│
└──────────────────────────────┘
```

#### 상단바 스펙

| 요소 | 스펙 |
|:---|:---|
| 높이 | 56px |
| 배경 | `--color-surface` (#FFFFFF) |
| 하단선 | `--color-border` 1px solid |
| position | `sticky; top: 0; z-index: 100` |
| 로고 | 좌측, 높이 32px, 탭 시 홈(/) 이동 |
| 검색 아이콘(🔍) | 우측, 48×48 터치 타겟, 탭 → 검색 오버레이 |
| 프로필 아이콘(👤) | 최우측, 48×48 터치 타겟 |

#### 프로필 아이콘 동작

| 상태 | 탭 동작 |
|:---|:---|
| 비로그인 | `/login` 이동 (카카오 로그인) |
| 로그인 | `/my` 마이페이지 이동 |

#### 아이콘 메뉴 행 스펙

| 요소 | 스펙 |
|:---|:---|
| 높이 | 64px |
| 배경 | `--color-surface` (#FFFFFF) |
| position | `sticky; top: 56px; z-index: 99` (상단바 바로 아래 붙음) |
| 하단선 | `--color-border` 1px solid |
| 아이콘 크기 | 28×28px |
| 터치 타겟 | 52×52px 이상 (아이콘+라벨 영역 전체) |
| 라벨 크기 | 12px (아이콘 동반이므로 예외 허용) |
| 아이템 간격 | 균등 배분 (`justify-content: space-around`) |
| 활성 상태 | 아이콘+라벨 `--color-primary`, 하단 2px indicator |
| 비활성 상태 | `--color-text-muted` (#999999) |
| 홈(/) 진입 시 | 모든 메뉴 비활성 (홈은 로고 탭으로 접근) |

### A3-3. 플로팅 글쓰기 버튼 (FAB)

> 사용자가 글을 쓸 수 있는 페이지에서만 표시.
> 시니어 사용자에게 "여기서 글을 쓸 수 있어요"를 직관적으로 안내.

```
                              ┌──────────┐
                              │ ✏️ 글쓰기 │  ← FAB (pill shape)
                              └──────────┘
```

#### FAB 표시 규칙

| 페이지 | FAB 표시 | 이유 |
|:---|:---:|:---|
| / (홈) | ❌ | 종합 대시보드, 글쓰기 맥락 없음 |
| /best | ❌ | 자동 집계 페이지 |
| /jobs, /jobs/[id] | ❌ | 운영자 전용 |
| **/community/stories** | **✅** | 사용자 UGC |
| **/community/humor** | **✅** | 사용자 UGC |
| /community/*/[id] (상세) | ❌ | 댓글로 참여 (별도 입력창) |
| /magazine, /magazine/[id] | ❌ | 운영자 전용 |
| /my/* | ❌ | 설정/관리 페이지 |

#### FAB 디자인

| 속성 | 모바일 | 데스크탑 |
|:---|:---|:---|
| 형태 | pill shape (아이콘 + "글쓰기" 텍스트) | 원형 56px (아이콘만), 호버 시 pill 확장 |
| 높이 | 52px | 56px |
| 위치 | 우하단, margin 24px | 우하단, margin 32px |
| 색상 | `--color-primary` (#FF6F61), 텍스트 #FFFFFF |
| 그림자 | `0 4px 12px rgba(0,0,0,0.15)` |
| z-index | 90 |
| 스크롤 동작 | 아래 스크롤 시 축소 (아이콘만), 위 스크롤/정지 시 확장 |
| 애니메이션 | scale + fade, 0.2s ease |

### A3-4. 글쓰기 UX 플로우

```
[사는 이야기 or 활력 충전소 목록]
    │
    ▼ FAB "✏️ 글쓰기" 탭
    │
    ├── 비로그인 ──→ 로그인 유도 바텀시트
    │                 "글을 쓰려면 로그인이 필요해요"
    │                 [카카오로 시작하기 🟡]
    │                 └── 로그인 완료 → 글쓰기 페이지
    │
    ├── 로그인 + 🌱새싹 이상 ──→ /community/write
    │     │
    │     ├── ① 게시판 선택 (자동: 진입한 게시판 프리셋)
    │     │     [사는 이야기 ○] [활력 충전소 ○]
    │     │
    │     ├── ② 제목 입력 (필수, 2~40자)
    │     │     placeholder: "제목을 입력해 주세요"
    │     │
    │     ├── ③ 본문 입력 (필수, 10자 이상)
    │     │     - 기본 서식: 볼드, 줄바꿈
    │     │     - 이미지 첨부: 🌿단골 이상만 활성 (비활성 시 "단골 등급부터 가능해요" 안내)
    │     │     - 유튜브 링크: 🌿단골 이상만 활성
    │     │     - 이미지: 최대 5장, 각 5MB 이하, WebP 자동 변환
    │     │
    │     ├── ④ 하단 액션바
    │     │     [취소] ─────────────────── [미리보기] [등록하기]
    │     │
    │     ├── 취소 탭 → "작성 중인 내용이 사라져요. 나가시겠어요?" 확인 모달
    │     │
    │     └── 등록 완료 → 작성한 글 상세 페이지로 이동 + 토스트 "글이 등록되었어요!"
    │
    └── 등급 미달 (비회원~가입 전) → 카카오 로그인 유도
```

### A5-6. 마이페이지 (/my)

> 로그인 필수. 내 활동 관리 + 설정.

#### 마이페이지 메인

```

│  가입일: 2026.03.10          │
│                              │
│  글 3개 · 댓글 12개 · 스크랩 5개 │ ← 활동 요약
├──────────────────────────────┤
│  📝 내가 쓴 글                │ → /my/posts
│  💬 내 댓글                   │ → /my/comments

│  🔔 알림                     │ → /my/notifications

├──────────────────────────────┤
│  🚪 로그아웃                  │

```

#### 설정 (/my/settings)

| 항목 | 내용 |
|:---|:---|
| 닉네임 변경 | 2~10자, 중복 체크, 30일 1회 제한 |
| 프로필 이미지 | 변경/삭제 (5MB 이하) |
| 알림 설정 | 댓글 알림 ON/OFF, 공감 알림 ON/OFF |
| 글자 크기 | [기본] [크게] [아주크게] → font-body 17/20/24px |
| 회원 탈퇴 | "탈퇴하면 모든 데이터가 삭제돼요" 확인 절차 |

#### 알림 (/my/notifications)

| 알림 유형 | 내용 |
|:---|:---|
| 댓글 알림 | "영희맘님의 글에 순자맘님이 댓글을 남겼어요" |
| 공감 알림 | "영희맘님의 글에 5명이 공감했어요" (묶음) |
| 등급 승격 | "축하해요! 🌿단골 등급이 되었어요" |
| 시스템 | 운영 공지, 이벤트 안내 |
| 글 상태 | "글이 숨김 처리되었어요" (신고 3회 시) |

## A6. UGC 시스템 — 댓글·등급·승격

### 댓글/대댓글 정책

| 규칙 | 내용 |
|:---|:---|
| 작성 | 🌱 새싹 이상 |
| 대댓글 | 1단계만 |
| 수정 | 10분 이내 |
| 삭제 | 본인 가능. 대댓글 있으면 "삭제된 댓글입니다" 표시 |
| 이미지 | 🌿 단골 이상 |
| 유튜브 | 🌿 단골 이상, URL 붙여넣기 자동 프리뷰 |
| 공감 | 모든 회원 ❤️ 가능 |
| 신고 | 3회 누적 → 자동 숨김 |
| 순서 | 등록순(기본) / 공감순 토글 |
| 필터 | AI 금지어 즉시 숨김 |
| 비회원 | 읽기만 |

### 등급별 권한

| 기능 | 비회원 | 🌱새싹 | 🌿단골 | 💎터줏대감 | ☀️따뜻한이웃 |
|:---|:---:|:---:|:---:|:---:|:---:|
| 열람·검색 | ✅ | ✅ | ✅ | ✅ | ✅ |
| 공감·스크랩·신고 | ❌ | ✅ | ✅ | ✅ | ✅ |
| 댓글 | ❌ | ✅ | ✅ | ✅ | ✅ |
| **글쓰기** | ❌ | **✅** | ✅ | ✅ | ✅ |
| **이미지·유튜브** | ❌ | ❌ | **✅** | ✅ | ✅ |

### 등급 조건

| 등급 | 조건 | 비고 |
|:---|:---|:---|
| 🌱 새싹 | 카카오 로그인 가입 완료 | 즉시 |
| 🌿 단골 | 글 5개 이상 **OR** 댓글 20개 이상 | 자동 승격 |
| 💎 터줏대감 | 글 20개 이상 **AND** 공감 100+ 받음 | 자동 승격 |
| ☀️ 따뜻한이웃 | PO 수동 부여 | 모범 사용자 |

### 글 승격 시스템

```
일반글 → 🔥 뜨는글 (공감 10+) → 👑 명예의전당 (공감 50+)
```

- 컷 수치: 어드민에서 조정 가능
- 비추천: 없음 (시니어 커뮤니티 특성상 부정적 피드백 최소화)
- 달성 시 영구 유지

### 로그인: 카카오톡 전용

```
[로그인 버튼 탭]
  → 카카오 OAuth 2.0
  → 최초 로그인 시:
     ├── 닉네임 설정 (2~10자, 필수)
     ├── 이용약관 동의 (필수)
     ├── 개인정보 처리방침 동의 (필수)
     └── 마케팅 수신 동의 (선택)
  → 가입 완료 → 🌱새싹 등급 자동 부여
  → 진입 전 페이지로 복귀
```

## A7. 광고 시스템 — 멀티소스 통합 설계

### 광고 유형 4가지

| 유형 | 설명 | 운영 주체 | 수익 |
|:---|:---|:---|:---|
| **자체 배너** | 이벤트, 설문, 매거진 홍보 등 | PO가 어드민에서 등록 | 간접 (유입) |
| **구글 애드센스** | 구글 자동 노출 배너/네이티브 광고 | 구글 자동 (코드 삽입) | CPC/CPM |
| **쿠팡 CPS** | 쿠팡파트너스 배너/상품 카드 | PO가 제휴 링크 등록 | CPS (구매 시 수수료) |
| **외부 직접** | 기업/기관에서 직접 의뢰한 광고 | PO가 어드민에서 등록 | 월정액/CPC |

### 슬롯별 설계

| 슬롯 ID | 위치 | 지원 유형 | 로테이션 |
|:---|:---|:---|:---|
| `HERO` | 홈 히어로 슬라이드 중 1장 | 자체/외부직접 | 슬라이드 내 순서 |
| `HOME-INLINE` | 홈 에디터스픽 아래 | 구글/쿠팡/자체/외부 | **우선순위 기반** |
| `SIDEBAR` | 데스크탑 우측 사이드바 | 구글/쿠팡/외부 | 랜덤 |
| `LIST-INLINE` | 게시글 목록 5번째/10번째 행 | 구글/쿠팡/자체 | 랜덤 |
| `POST-BOTTOM` | 글 상세 본문 하단 | 구글/쿠팡/자체 | 우선순위 |
| `MOBILE-STICKY` | 모바일 화면 하단 고정 | 구글/외부 | 고정 |
| `MAGAZINE-CPS` | 매거진 글 하단 상품 영역 | **쿠팡 CPS 전용** | 관련 상품 |

### 우선순위 로직

```
슬롯에 여러 광고가 등록된 경우:
1순위: 외부 직접 (유료, 계약 기반)
2순위: 자체 배너 (이벤트, 홍보)
3순위: 쿠팡 CPS (수익)
4순위: 구글 애드센스 (기본 fill)

→ 외부 직접 광고가 있으면 그것 노출
→ 없으면 자체 배너
→ 없으면 쿠팡
→ 아무것도 없으면 구글 애드센스 자동 fill
```

### 광고 표시 규칙

| 규칙 | 내용 |
|:---|:---|
| 라벨 | 모든 광고에 "광고" 또는 "AD" 라벨 필수 |
| 배경 | `--color-ad-bg` (#F9F5F0) 구분 |
| 빈도 | 모바일: 화면당 최대 1개 광고 (과도한 광고 방지) |
| 금지 | 정치/도박/성인/약물/다단계 광고 **절대 금지** |
| 닫기 | 모바일 스티키는 ✕ 닫기 버튼 제공 (24시간 안 보기) |
| CPS 투명성 | 쿠팡 링크에는 항상 안내 문구 표시 |

## A8. 공통 UX 패턴

### 공유 기능

| 플랫폼 | 방법 |
|:---|:---|
| 카카오톡 | 카카오 SDK 공유 (썸네일 + 제목 + 링크) |
| 링크 복사 | 클립보드 복사 + 토스트 "링크가 복사되었어요" |
| (Phase 2) | 네이버 밴드 공유 추가 고려 |

### 빈 상태 (Empty State)

| 페이지 | 메시지 | CTA |
|:---|:---|:---|
| 검색 결과 없음 | "검색 결과가 없어요" | "다른 키워드로 검색해 보세요" |
| 게시글 목록 비어있음 | "아직 글이 없어요" | "첫 번째 글을 써보세요! [글쓰기]" |
| 내가 쓴 글 없음 | "아직 작성한 글이 없어요" | "첫 글을 남겨보세요! [글쓰기]" |
| 스크랩 없음 | "스크랩한 글이 없어요" | "마음에 드는 글에 📌 눌러보세요" |
| 알림 없음 | "새 알림이 없어요" | (없음) |

### 로딩 상태

| 유형 | 구현 |
|:---|:---|
| 페이지 전환 | 상단 프로그레스 바 (NProgress 스타일) |
| 카드 로딩 | 스켈레톤 UI (카드 모양 그레이 블록) |
| 이미지 로딩 | 저화질 blur 프리뷰 → 고화질 전환 |
| 버튼 로딩 | 버튼 내 스피너 + 비활성화 |

### 에러 상태

| 유형 | 메시지 | 액션 |
|:---|:---|:---|
| 네트워크 오류 | "인터넷 연결을 확인해주세요 📶" | [다시 시도] |
| 서버 오류 | "잠시 문제가 생겼어요. 곧 돌아올게요!" | [다시 시도] |
| 404 | "앗, 이 페이지를 찾을 수 없어요" | [홈으로 가기] |
| 권한 없음 | "로그인이 필요한 페이지예요" | [카카오로 로그인] |

### 토스트 메시지

| 상황 | 메시지 | 지속시간 |
|:---|:---|:---|
| 글 등록 완료 | "글이 등록되었어요!" | 3초 |
| 댓글 등록 | "댓글이 등록되었어요" | 2초 |
