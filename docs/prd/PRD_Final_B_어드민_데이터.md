### 콘텐츠 관리

**테이블 열**: ☐ | ID | 게시판(뱃지) | 제목(클릭→고객웹) | 작성자(봇=🤖) | 소스(BOT/USER/ADMIN) | 상태 | ❤️ | 💬 | 👁 | 승격(—/🔥/👑) | 등록일 | 액션[보기·수정·숨김·삭제·핀]

**필터**: 게시판 / 상태 / 소스 / 기간 / 검색(제목+본문+작성자)

**일괄 액션**: [숨김] [삭제] [게시판이동] [핀설정] [핀해제]

### 에디터스 픽

현재 핀 리스트 (최대 3) + 순서 드래그 + 해제 + 추가(콘텐츠 검색 모달)

### 봇 관리

각 봇: 마지막실행 / 다음실행 / 오늘 결과(수집→필터→발행) / 7일 통계 / [로그보기] [지금실행]

**검수 큐**: 봇+제목+필터사유+원문미리보기 → [✅승인] [❌거부]

### 모더레이션

**신고 처리 워크플로우**: 접수→대기→[삭제/숨김/경고/정지/차단/기각]→완료. 처리 시 사용자 알림 자동.

**제재 4단계**: 경고 → 정지7일 → 정지30일 → 영구차단. 마이페이지에 상태 표시.

**금지어**: 카테고리분류 + CSV 일괄 추가 + AI 문맥 판단(바람피다 vs 봄바람) + 필터링 이력 로그

### 배너·광고 관리

**히어로**: 슬라이드 최대 3장, 드래그 순서, 기간, 미리보기(모바일/데스크탑)

**광고 슬롯**: 7개 슬롯 탭별 관리. 각 슬롯에 **여러 광고 등록 가능** (우선순위 기반 노출).

| 설정 | 상세 |
|:---|:---|
| 광고 유형 | 자체/구글/쿠팡/외부 선택 |
| 이미지 or HTML | 업로드 or 코드 입력 |
| 클릭 URL | 외부 링크 |
| 기간 | 시작~종료 |
| 우선순위 | 숫자 (낮을수록 먼저) |
| 통계 | 노출/클릭/CTR 자동 계산 |

### 회원 관리

목록 테이블 + 상세 페이지(활동이력·제재이력·신고이력) + 등급변경 + 따뜻한이웃 수여

### 매거진 에디터

WYSIWYG + AI초안 + CPS삽입 + SEO편집 + 예약발행 + 초안목록 + 미리보기
