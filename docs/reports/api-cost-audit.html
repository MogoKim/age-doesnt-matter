3. API 장애 시 상세 영향

ℹ️
각 API가 완전히 중단되면 어떤 일이 벌어지는지 정리. 우선순위 높은 항목부터 확인.
API즉각 영향 (0~1h)누적 영향 (1일+)사용자 체감자동 복구
Supabase DB 모든 페이지 500. 로그인 불가. 에이전트 전부 중단 서비스 완전 마비 사이트 자체가 열리지 않음 ❌
Anthropic Claude 에이전트 전체 콘텐츠 생성 중단. Slack #긴급 알림 매거진 공백, 시드봇 조용, SNS 포스팅 중단 "새 글이 안 올라온다" ❌ 충전 필요
Slack 알림 전송 실패. 에이전트는 계속 실행됨 오류 발생해도 모름 — 모니터링 블라인드 내부적으로만. 사용자는 정상처럼 느낌 ❌
Cloudflare R2 이미지/영상 업로드 실패. 기존 이미지는 정상 신규 매거진 이미지 없음, 프로필 사진 업로드 불가 매거진 이미지 깨짐 (신규 글만) ❌
Instagram/Threads SNS 포스팅 실패. BotLog FAILED 소셜 채널 침묵 SNS에 글이 안 올라옴 ❌ 재시도 없음
X (Twitter) 429 에러 시 즉시 실패 X 채널 침묵 X에 글이 안 올라옴 ❌ 재시도 없음
Google Indexing 색인 요청 실패 검색 노출 지연 (며칠~1주) 신규 글이 검색에 잘 안 잡힘 ❌
OpenAI DALL-E 이미지 생성 실패 → Unsplash 폴백 매거진 이미지 품질 하락 이미지가 일관성 없어짐 ✅ Unsplash 폴백
Perplexity AI 트렌드 리서치 실패 Claude만으로 트렌드 생성 매거진 주제가 덜 트렌디 ✅ Claude 폴백
Google GA4 CDO/CEO 리포트 데이터 없음 KPI 대시보드 공백 Slack 리포트에 데이터 없음 ✅ null 폴백
Kakao OAuth 신규 로그인 불가. 기존 세션은 유지 로그아웃 후 재로그인 불가 "카카오 로그인이 안 됩니다" ❌ 카카오 측 복구 필요
Whisper / Gemini TTS 영상 자막/나레이션 생성 실패 (수동이라 즉각 영향 없음) 무음 또는 자막 없는 영상 영상 품질 하락 수동 프로세스
Facebook / Band 현재 비활성화 상태 — 영향 없음

4. CFO 자동 차단 미구현 — 구조적 공백

🚨
헌법에는 "$50 초과 시 즉시 잠금" 정책이 있지만 cost-tracker.ts에 실제 구현 코드가 없다. $50 초과해도 에이전트는 계속 실행된다. 정책은 선언문일 뿐, 집행자가 없다.

실제 발생한 일

일 $3.77 소비 × 7일 = $26.39 → $27.50 grant 소진 → -$0.17 초과 → Claude API 중단
CFO는 감지했겠지만 자동으로 아무것도 하지 않았다.

해결: cost-tracker.ts에 추가해야 할 코드

if (monthlyCost >= MONTHLY_HARD_LIMIT) {
  await db.systemSetting.upsert({
    where: { key: 'automation_status' },
    update: { value: 'LOCKED' },
    create: { key: 'automation_status', value: 'LOCKED' }
  });
  await notifySlack('system',
    `🚨 월 비용 $${monthlyCost} 초과 → 에이전트 자동 잠금`
  );
}

6. 비상 정지 방법

# 1. 에이전트 전체 잠금 (모니터링 5개만 유지)
# agents/core/constitution.yaml 수정:
automation_status: "LOCKED"
# → $74/월 → $1/월 즉시 전환

# 2. 개별 workflow 비활성화
gh workflow disable agents-daily.yml
gh workflow disable agents-social.yml
gh workflow disable agents-cafe.yml
gh workflow disable agents-seed.yml

# 3. 상태 확인
gh workflow list

LOCKED 시 유지되는 에이전트: cto:health-check / error-monitor / security-audit / qa-verify / cdo:anomaly-detector
