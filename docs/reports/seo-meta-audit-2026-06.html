1우리는 누구를 위한 서비스인가

모든 제목·설명·이미지는 결국 "이 사람"에게 가닿아야 합니다. 그 기준이 흔들리면 문구도 흔들립니다.

코드에 공식으로 박혀 있는 타겟 정의입니다 (추측 아님, 창업자 확정 기록):

constitution.yaml (창업자 확정 2026-04-13) target.primary : "40대 후반~60대 여성 (100%)" target.secondary: null ← 남성 타깃 제거

실제 페르소나 5명도 전원 여성입니다: 영숙(58)·정희(63)·미영(55)·순자(61)·현주(57).

👉 그래서 이 문서의 채점 기준은 하나입니다
"지금 화면에 40대 후반~60대 여성이라는 우리 정체성이 잘 보이는가?"

구성: og:image(그림) · og:title(제목) · og:description(설명) · og:url(주소). 권장 그림 크기는 가로 1200×630입니다.

5문제 정의 (우선순위)

채점 기준은 "40대 후반~60대 여성"이 잘 보이나. 이 기준으로 P0(급함)→P2(나중) 정렬.
🔴 가장 큰 모순 — 여성 정체성이 "검색되는 곳"에서 사라짐
여성 카피가 있는 페이지 ─→ landing ─→ 🚫 검색에서 숨김(noindex) 검색·공유로 보이는 페이지 ─→ 홈·목록·글 ─→ 🟡 "5060 세대"(여성 없음)
즉 "여성"이라 써둔 페이지는 검색에 안 뜨게 막혀 있고, 정작 사람들이 보는 페이지엔 여성이 빠져 있습니다.
P0 · 문제 1-1 / 2-1 / 4-1
검색·공유 메타에 여성 정체성이 전무
홈·목록·게시판 설명 전부 "5060 세대"(성별 중립). 우리 핵심 타겟이 가장 많이 보이는 화면에서 안 보입니다.
P0 · 문제 3-1 / 3-2 / 3-3
공유 명함(OG)이 깨짐
기본 이미지가 정사각형이라 가로 카드에서 양옆 잘림 + 카피·정체성 없는 로고 한 장 + 카카오 버튼 fallback 그림은 아예 404로 안 뜸.
P1 · 문제 1-2 / 4-2
목록·게시판 제목·설명이 밋밋
"인기글", "매거진"처럼 기능 설명만. 검색결과에서 차별성·정체성이 약합니다.
참고 · 5-1 (문제 아님으로 정정)
커뮤니티 글은 본문 발췌가 검색 설명으로 나감 — 이게 정상
UGC(사용자·커뮤니티 글)는 인위적 SEO 설명보다 본문 발췌가 더 적합(네이버카페·레딧 방식). 매거진만 AI SEO 최적화, 커뮤니티는 본문 노출 — 이미 합리적으로 분리됨. 유일한 보강거리는 "발췌 첫 문장이 잡담 없이 맥락 있게 시작하나"(발췌 품질, SEO 아님).
P2 · 문제 6-1~3
서비스 이름 3개 + 트로트 곡명 충돌
"우리 나이가 어때서" / "우나어" / 영어도메인 혼용. 게다가 동명의 트로트 곡 때문에 검색 시 노래가 먼저 뜸.
🟢 반대로, 이미 잘 되어 있는 것 (손댈 필요 없음)
  • Robots(검색 노출 제어) — 보일·숨길 페이지 분리 완성
  • 글·일자리 상세의 자동 공유 명함(동적 OG 1200×630)
  • 매거진 글의 AI 자동 SEO(제목·설명에 "50대" 키워드 포함)
