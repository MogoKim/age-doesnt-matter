우나어 — 근본을 파헤치다

이 문서는 빈 캔버스이자 실행 지도다. 창업자가 각 섹션을 딥다이브로 채울 때, 시스템 전체에 어떤 산출물이 바뀌어야 하는지를 자동으로 보여준다.

크롤링은 수동으로 직접 돌린다. 양보다 질 — 5060이 가장 뜨겁게 반응한 게시물을 빡세게 긁어낸 뒤, 반응 지수(좋아요+댓글×2+조회수×0.1)로 정렬해 욕망을 귀납한다.

매출 공식 (Revenue Formula)
"돈은 어떤 구조로 벌리는가?"
⚠️ 구조 있음, 드라이버 필요
월 매출 = DAU × PV/DAU × eCPM/1000 × 30일 + CPS 수익 월 순수익 = 월 매출 - (Vercel + Supabase + Claude API + 기타) 현재 운영 비용 예상: · Vercel (Pro): ~$20/월 · Supabase (Pro): ~$25/월 · Claude API (매거진봇 등): ~$10-50/월 ───────────────────────────────────── 합계: 약 $55~95/월 (≈ 8~14만원)
각 변수의 드라이버 (채워야 할 것)
  • DAU 드라이버: SEO 유입 × 재방문율(RR) × [콘텐츠 신선도, 알림, 플라이휠]
  • PV/DAU 드라이버: 내부 링크 구조, 관련글 추천, 댓글 알림 클릭, 카테고리 탐색
  • eCPM 드라이버: 콘텐츠 주제(건강/돈 관련이 eCPM 높음), 페이지 체류시간
  • CPS 드라이버: 쿠팡파트너스 링크 클릭 유도 시점, 건강/생활용품 콘텐츠 비율
  • eCPM vs CPC 구조 파악: 애드센스 실제 수익 모델 확인 필요
