## 2. 뭘 개발했나? (코드 1,347줄)

### 2-1. 일자리 자동 수집 시스템 (Phase 1)

**이게 뭔가요?**
매일 3번(낮 12시, 오후 4시, 저녁 8시) 자동으로 50plus.or.kr에서 시니어 일자리를 수집해서
우나어 사이트에 게시합니다. 사람이 아무것도 안 해도 됩니다.

**작동 과정:**
```
50plus.or.kr 접속 (자동)
  → 공고 30개 수집
  → DB에서 이미 올린 거 제거 (중복 방지)
  → 좋은 공고 4~5개 엄선 (아래 기준으로)
  → AI가 제목/설명/Q&A 다듬기
  → 우나어 DB에 저장 → 사이트에 자동 노출
  → 텔레그램으로 "몇 건 게시했어요" 알림
```

**엄선 기준 (예전 아임웹 시스템에서 가져옴):**
- 1순위: 대기업/공공기관 일자리
- 2순위: 사무직/안내/상담 등 좋은 조건
- 3순위: 지방 광역시 일자리 (지역 균형)
- 4순위: 나머지
- 여성 친화 일자리 60% 이상 유지
- 수도권 50% / 지방 50% 균형

**만든 파일 4개:**

| 파일 | 줄 수 | 역할 |
|------|-------|------|
| `agents/coo/job-scraper.ts` | 352줄 | 50plus.or.kr 크롤링 + 전체 파이프라인 |
| `agents/coo/job-processor.ts` | 249줄 | AI가 제목/SEO키워드/포인트/Q&A 생성 |
| `agents/coo/job-filter.ts` | 149줄 | Waterfall 필터링 + 쿼터 로직 |
| `agents/coo/job-types.ts` | 104줄 | 타입 정의 |
