🔴 3. 문제점 — 지금 안 되는 것

  • 에이전트 간 실시간 협업 부재 — 각자 cron으로 독립 실행될 뿐, 서로 소통/반응 구조 없음
  • 크롤링 인사이트 → 콘텐츠 단절 — 카페 트렌드가 SEED 글쓰기에 자동 반영 안 됨
  • 크롤러 품질 문제 — 가입인사 같은 무의미한 글까지 수집, 필터링/전략 부재
  • 데이터 → 자동 가설 수립 부재 — CDO가 수집해도 병목→가설→실험 자동화 안 됨
  • 채널별 전담 에이전트 없음 — CMO 산하 채널별 세분화 필요
  • 실험 주기 자동화 부재 — 주 1~2회 사이클이 수동 개입 없이 안 돌아감
  • Slack 승인 누락 위험 — 한 채널에 로그+승인 혼재하여 중요 건 놓침
